# 创意发掘

## 工作流程

### 第一步：定义筛选条件

向用户确认参数：
- **方向**：多头创意、空头创意，或两者兼有
- **市值**：大盘、中盘、小盘、微盘
- **行业**：特定行业或跨行业
- **风格**：价值、成长、质量、特殊情况、事件驱动
- **地域**：美国、国际、全球
- **主题**：任何特定主题角度（AI、回流、老龄化等）

### 第二步：量化筛选

根据风格运行筛选：

**价值筛选**
- P/E低于行业中位数
- EV/EBITDA低于历史均值
- 自由现金流收益率 >5%
- 市净率低于1.5倍
- 过去90天内部人买入
- 股息率高于市场均值

**成长筛选**
- 收入同比增长 >15%
- 盈利同比增长 >20%
- 收入加速（增长率上升）
- 利润率扩张
- 高投入资本回报率（>15%）
- 强劲的净留存率（SaaS >110%）

**质量筛选**
- 持续收入增长（5年以上）
- 稳定或扩张的利润率
- ROE >15%
- 低负债权益比
- 高自由现金流转化率
- 内部人持股 >5%

**空头筛选**
- 收入下降或增长减速
- 利润率压缩
- 应收账款/存货相对于销售上升
- 内部人卖出
- 相对同行无合理理由的估值溢价
- 高做空兴趣且基本面恶化
- 会计红旗（审计师变更、重述）

**特殊情况筛选**
- 近期IPO/SPAC锁定期到期
- 过去12个月内的分拆
- 重组后恢复的公司
- 激进投资者介入
- 业绩不佳公司的管理层变动

### 第三步：主题扫描

对主题性创意，研究主题并识别受益者：

1. 定义论点（例如，"AI基础设施支出在2026年前加速"）
2. 绘制价值链——谁直接 vs 间接受益？
3. 识别纯主题标的 vs 多元化敞口
4. 评估哪些标的已被"price in" vs 被低估
5. 寻找市场尚未与该主题关联的二阶受益者

### 第四步：创意呈现

对通过筛选的每个创意，呈现：

**[公司名称] — [多头/空头] — [一句话论点]**

| 指标 | 数值 | vs 同行 |
|------|------|---------|
| 市值 | | |
| EV/EBITDA (NTM) | | |
| P/E (NTM) | | |
| 收入增长 | | |
| EBITDA利润率 | | |
| FCF收益率 | | |

**论点（3-5条）：**
- 为什么被错误定价
- 市场忽略了什么
- 实现价值的催化剂

**主要风险：**
- 什么会使这个论点失效

**建议下一步：**
- 建立完整模型？深度尽调？专家访谈？

### 第五步：输出

- 5-10个创意的短名单，附一页纸摘要
- 筛选标准和方法论文档
- 所有创意的比较表
- 优先级列表：哪些创意优先研究

## 重要提示

- 筛选是发掘候选标的，而非结论——每个筛选输出都需要基本面研究
- 最佳创意往往来自交叉点（例如，因暂时性逆风而处于价值价格的优质公司）
- 避免拥挤交易——检查持股数据、做空兴趣和覆盖分析师数量
- 逆势创意需要催化剂——没有催化剂的提前布局等同于错误
- 跟踪创意命中率——哪些筛选和方法产生最佳创意？
- 空头创意需要更高确信度——择时更难且风险不对称
