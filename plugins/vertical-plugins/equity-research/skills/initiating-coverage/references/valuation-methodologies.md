## 贴现现金流（DCF）分析

### 概述

DCF 分析基于公司预计未来现金流的现值来对公司进行估值。这被认为是最具理论基础的估值方法，因为它基于基本价值创造。

### 分步 DCF 流程

#### 1. 历史财务分析
- 收集 3-5 年历史财务数据
- 计算历史自由现金流 = EBIT(1-税率) + 折旧摊销 - 资本支出 - 营运资金变动
- 分析历史增长率和利润率
- 识别趋势和周期性

#### 2. 构建收入预测（5-10 年）
**方法：**
- **自上而下**：从市场规模（TAM）开始 → 市场份额 → 收入
- **自下而上**：销量 × 单价
- **混合**：结合多个驱动因素

**关键考量：**
- 管理层指引和历史增长
- 行业增长率和市场趋势
- 竞争动态和市场份额演变
- 产品管线和新市场机会
- 宏观经济因素

#### 3. 预测运营费用
- **销售成本**：占收入百分比（分析历史利润率）
- **销售及管理费用**：通常为半固定；按收入百分比建模，考虑规模效应
- **研发费用**：对科技/制药公司至关重要；按收入百分比建模
- **折旧摊销**：基于资本支出假设

**计算 EBIT** = 收入 - 销售成本 - 运营费用

#### 4. 计算无杠杆自由现金流
```
EBIT
× (1 - 税率)
= 税后净营业利润（NOPAT）
+ 折旧与摊销
- 资本支出
- 净营运资金增加
= 无杠杆自由现金流（UFCF）
```

**资本支出假设：**
- 维护性资本支出：维持当前运营所需（通常为收入的 2-4%）
- 增长性资本支出：扩张所需
- 考虑行业基准和公司指引

**净营运资金：**
- 净营运资金 =（应收账款 + 存货）- 应付账款
- 按收入百分比或天数建模（应收账款周转天数、存货周转天数、应付账款周转天数）
- 净营运资金增加是现金的使用

#### 5. 确定终值

**方法 A：永续增长法**
```
终值 = 最后一年自由现金流 × (1 + g) / (WACC - g)
```
- g = 永续增长率（通常 2-3%，不超过 GDP 增长率）
- 当公司已达到稳定、成熟增长时使用

**方法 B：退出倍数法**
```
终值 = 最后一年 EBITDA × 退出倍数
```
- 退出倍数基于当前可比交易
- 更适用于周期性行业

#### 6. 计算加权平均资本成本（WACC）

```
WACC = (E/V × 股权成本) + (D/V × 债务成本 × (1 - 税率))
```

**股权成本（使用 CAPM）：**
```
股权成本 = 无风险利率 + Beta × 股权风险溢价
```
- 无风险利率：10 年期国债收益率
- Beta：股票收益率与市场的回归（或使用可比公司 Beta）
- 股权风险溢价：历史平均约 5-6%

**债务成本：**
```
债务成本 = 无风险利率 + 信用利差
```
- 使用公司当前借款利率或从债券推算
- 如无未偿还债券，根据信用评级调整

**资本结构：**
- E/V = 股权市场价值 / 总价值
- D/V = 债务市场价值 / 总价值
- 使用目标资本结构，而非当前（如差异显著）

#### 7. 将现金流贴现至现值

```
现值 = Σ [FCFt / (1 + WACC)^t] + [终值 / (1 + WACC)^n]
```

#### 8. 计算企业价值和股权价值

```
企业价值 = 预计自由现金流现值 + 终值现值
减：净债务（总债务 - 现金）
加：非经营性资产
减：少数股东权益
减：优先股
= 股权价值

每股价格 = 股权价值 / 稀释后流通股数
```

### DCF 敏感性分析

始终对关键变量进行敏感性分析：

1. **双向敏感性表**：WACC vs. 永续增长率
2. **收入增长情景**：基准/乐观/悲观
3. **利润率假设**：经营杠杆情景
4. **终值倍数敏感性**：如使用退出倍数法

**敏感性表示例：**
```
           永续增长率
WACC      2.0%    2.5%    3.0%
8.0%      45元     48元     52元
9.0%      40元     43元     46元
10.0%     36元     39元     41元
```

### 需避免的常见 DCF 陷阱

1. **重复计算增长**：不要在没有相应投资（资本支出、净营运资金）的情况下预测高增长
2. **不切实际的永续增长**：不应超过长期 GDP 增长率
3. **忽视周期性**：对周期性行业正常化盈利
4. **错误的现金流定义**：使用无杠杆自由现金流，而非净利润
5. **假设不一致**：贴现率与现金流匹配（无杠杆自由现金流 → WACC）

## 可比公司分析

### 概述

可比公司分析基于类似公司在公开市场的估值来对公司进行估值。这反映了当前市场情绪和相对估值。

### 分步可比公司流程

#### 1. 选择可比公司

**选择标准：**
- 相同行业/板块（首要标准）
- 类似的商业模式和收入来源
- 可比规模（市值、收入）
- 类似的增长特征和利润率
- 类似的终端市场和地理覆盖

**典型范围：**
- 从 8-15 家公司开始
- 剔除有特殊情况的公司
- 最终 5-10 家公司

#### 2. 收集财务信息

**所需数据：**
- 当前股价和流通股数
- 最新财年财务报表
- 来自一致预期的下一年（NTM）预估
- 历史增长率

**计算市场指标：**
- 市值 = 股价 × 流通股数
- 企业价值 = 市值 + 债务 + 少数股东权益 + 优先股 - 现金
- 净债务 = 总债务 - 现金及等价物

#### 3. 计算估值倍数

**企业价值倍数：**
- **EV/Revenue**：适用于早期/高增长公司
- **EV/EBITDA**：最常用；适用于资本密集型企业
- **EV/EBIT**：当折旧摊销差异较大时有用

**股权价值倍数：**
- **P/E（市盈率）**：使用最广泛
- **P/B（市净率）**：适用于金融机构
- **P/S（市销率）**：适用于亏损公司

**计算口径：**
- 过去十二个月（LTM）- 历史
- 未来十二个月（NTM）- 前瞻性（首选）

#### 4. 分析并选择倍数

**创建可比公司表：**

| 公司 | 市值 | EV/Revenue | EV/EBITDA | EV/EBIT | P/E (NTM) | 收入增长 | EBITDA 利润率 |
|---------|-----------|------------|-----------|---------|-----------|----------------|---------------|
| 可比 A  | 100亿元      | 3.5x       | 12.0x     | 18.0x   | 22.0x     | 15%            | 28%           |
| 可比 B  | 80亿元       | 3.0x       | 10.5x     | 16.0x   | 19.0x     | 12%            | 27%           |
| ...     | ...       | ...        | ...       | ...     | ...       | ...            | ...           |
| 中位数  | -         | **3.2x**   | **11.0x** | **17.0x** | **20.5x** | 13%            | 27.5%         |

**调整：**
- 剔除异常值（通常超过 2 个标准差）
- 考虑使用中位数而非均值（受异常值影响较小）
- 如某些可比公司更具可比性，可加权
- 针对增长、利润率、风险差异进行调整

#### 5. 将倍数应用于目标公司

**计算示例：**
```
目标公司 NTM EBITDA = 500M元
选定 EV/EBITDA 倍数 = 11.0x
隐含企业价值 = 500M元 × 11.0x = 5,500M元

减：净债务 = 1,000M元
股权价值 = 4,500M元

流通股数 = 100M
隐含每股价格 = 45元.00
```

#### 6. 选择适当倍数

**选择依据：**
- **EV/Revenue**：高增长、亏损公司（科技、生物技术盈利前）
- **EV/EBITDA**：最常用；资本密集型行业（制造、电信）
- **P/E**：盈利稳定、资本结构稳定的公司（消费、零售）
- **行业特定**：银行用 P/B，油气用 EV/产量，媒体用 EV/用户数

### 溢价/折价分析

根据以下因素应用溢价或折价：
- **增长溢价**：增长越高 → 倍数越高
- **盈利能力**：利润率越高 → 倍数越高
- **规模**：大公司通常享有溢价（流动性）
- **市场地位**：市场领导者 → 溢价
- **地理因素**：发达市场 vs. 新兴市场

## 可比交易分析

### 概述

可比交易分析基于并购交易中为类似公司支付的价格来对公司进行估值。这反映了控制权溢价和战略价值。

### 分步流程

#### 1. 识别相关交易

**选择标准：**
- 相同或类似行业
- 类似规模（在目标公司 0.5-2 倍范围内）
- 类似的业务特征
- 近期交易（优选过去 3-5 年）
- 已公布并完成的交易（避免已撤回的交易）

**典型范围：**
- 至少 5-10 笔交易
- 重点关注近期交易（近期交易权重更高）

#### 2. 收集交易详情

**所需信息：**
- 交易日期（公布和完成）
- 收购价格和结构（现金、股票、混合）
- 交易时目标公司的财务数据
- 战略理由和协同效应
- 支付的控制权溢价

**来源：**
- 上市公司公告（重大资产重组报告书、临时公告、股东大会委托书）
- 新闻稿和投资者演示文稿
- 并购数据库（Wind、iFinD、Choice）

#### 3. 计算交易倍数

**与可比公司相同的倍数，但基于交易价值：**

```
交易价值 = 股权收购价格 + 承担债务 - 收购的现金

EV/Revenue (LTM) = 交易价值 / 目标公司 LTM 收入
EV/EBITDA (LTM) = 交易价值 / 目标公司 LTM EBITDA
EV/EBIT (LTM) = 交易价值 / 目标公司 LTM EBIT
```

**计算控制权溢价：**
```
控制权溢价 = (收购价格 - 未受影响价格) / 未受影响价格
```
- 未受影响价格 = 公布前 1-2 天目标公司股价
- 典型范围：20-40%

#### 4. 分析可比交易

**创建可比交易表：**

| 日期 | 目标公司 | 收购方 | 交易价值 | EV/Revenue | EV/EBITDA | 溢价 | 战略理由 |
|------|--------|----------|------------|------------|-----------|---------|-----------|
| 2024年一季度 | CompX | BuyerA | 50亿元 | 4.0x | 14.0x | 35% | 市场整合 |
| 2023年三季度 | CompY | BuyerB | 35亿元 | 3.5x | 12.5x | 28% | 战略契合 |
| 中位数 | - | - | - | **3.8x** | **13.0x** | **31%** | - |

#### 5. 应用于目标公司

**重要考量：**
- 可比交易倍数通常高于可比公司（包含控制权溢价）
- 针对交易战略理由的差异进行调整
- 考虑交易时的市场条件与当前对比
- 近期交易权重更高

**计算示例：**
```
目标公司 LTM EBITDA = 45元0M
选定 EV/EBITDA 倍数 = 13.0x（可比交易）
vs. 可比公司倍数 = 11.0x

隐含企业价值（可比交易）= 45元0M × 13.0x = 5,850M元
隐含企业价值（可比公司）= 45元0M × 11.0x = 4,950M元

隐含控制权溢价 = 5,850M元 / 4,950M元 - 1 = 18%
```

### 交易倍数调整

**考虑调整因素：**
- **市场条件**：牛市 vs. 熊市（并购活动水平）
- **交易结构**：战略收购方 vs. 财务收购方
- **协同效应**：高协同效应交易享有溢价
- **竞争动态**：单一竞标者 vs. 多个竞标者
- **时间价值**：较早的交易相关性较低

## 估值协调

### 创建估值桥接

将三种方法呈现在单一框架中：

**估值汇总示例：**

| 方法 | 企业价值 | 股权价值 | 每股价格 | 权重 | 隐含价值 |
|--------|------------------|--------------|-------------|--------|---------------|
| DCF 分析 | 5,200M元 | 4,200M元 | 42.00元 | 50% | 21.00元 |
| 可比公司 | 5,500M元 | 4,500M元 | 45元.00 | 30% | 13.50元 |
| 可比交易 | 5,850M元 | 4,850M元 | 48.50元 | 20% | 9.70元 |
| **加权平均** | - | - | **44.20元** | - | **44.20元** |

### 方法权重

**典型权重：**
- **DCF**：40-60%（基本价值，但假设预测准确性）
- **可比公司**：25-40%（反映当前市场情绪）
- **可比交易**：15-25%（除非可能发生并购，否则相关性较低）

**根据以下因素调整权重：**
- **对预测的信心**：信心越高 → DCF 权重越高
- **市场条件**：牛/熊市影响可比公司可靠性
- **并购可能性**：如公司处于并购中或行业整合，权重更高
- **公司成熟度**：成熟公司 → 可比公司权重更高；成长型 → DCF 权重更高

### 估值区间

始终呈现估值区间，而非点估计：

**方法：**
- **基准情景**：最可能的情景
- **乐观情景**：乐观假设（收入增长、利润率）
- **悲观情景**：保守假设

**示例：**
```
悲观情景：38元 - 40元
基准情景：42元 - 46元
乐观情景：48元 - 52元

评级：买入，目标价 45元（基准情景中点）
```

### 合理性检查

**交叉验证估值：**
1. **历史倍数**：当前估值是否与历史一致？
2. **同行比较**：相对同行的溢价/折价是否合理？
3. **隐含增长**：市场定价了什么增长？
4. **隐含回报**：从当前价格到目标价格的内部收益率
5. **市值分析**：总市值是否合理？

---

## 结论

使用所有三种估值方法为确定公允价值提供了稳健的框架：

- **DCF** 提供基于基本面的内在价值
- **可比公司** 反映当前市场估值
- **可比交易** 指示并购价值和控制权溢价

关键在于理解驱动每种方法的假设，并呈现考虑多种情景和方法论的合理估值区间。
