### 步骤2：构建 DCF 分析

#### A. 计算 WACC

**1. 确定无风险利率**
   - 使用10年期国债收益率（查看当前利率）
   - 示例：2024年末约4.0-4.5%

**2. 确定权益成本（CAPM）**
   ```
   权益成本 = 无风险利率 + Beta × 权益风险溢价

   输入：
   - 无风险利率：[当前10年期国债，如4.2%]
   - Beta：[来自 Wind / iFinD 的公司 Beta 或同行平均值]
   - 权益风险溢价：5-6%（历史平均值）

   示例：
   权益成本 = 4.2% + 1.3 × 5.5% = 11.35%
   ```

**3. 确定债务成本**
   ```
   债务成本 = 当前借款利率或债券隐含收益率

   非上市公司：
   债务成本 = 无风险利率 + 信用利差（基于评级）

   示例：
   债务成本（税前）= 6.5%
   债务成本（税后）= 6.5% × (1 - 25%税率) = 4.875%
   ```

**4. 确定资本结构**
   ```
   使用市场价值（非账面价值）：

   权益市场价值 (E) = 股价 × 流通股数
   债务市场价值 (D) = 总债务（如债券未交易则使用账面价值）
   总价值 (V) = E + D

   权益权重 = E / V
   债务权重 = D / V

   示例：
   E = 5,000M元 (90.9%)
   D = 500M元 (9.1%)
   V = 5,500M元 (100%)
   ```

**5. 计算 WACC**
   ```
   WACC = (E/V × 权益成本) + (D/V × 债务成本 × (1 - 税率))

   示例：
   WACC = (90.9% × 11.35%) + (9.1% × 6.5% × (1 - 25%))
   WACC = 10.32% + 0.44% = 10.76%

   基准情景取整：10.8%
   ```

#### B. 计算终值

**方法1：永续增长法（首选）**
```
终值 = FCF(2029) × (1 + g) / (WACC - g)

其中：
- FCF(2029) = 模型中最后一年的无杠杆 FCF
- g = 永续增长率（通常2.0-3.0%）
  - 不应超过长期 GDP 增速
  - 基准情景使用2.5%

示例：
FCF(2029) = 500M元
g = 2.5%
WACC = 10.8%

终值 = 500M元 × (1.025) / (0.108 - 0.025)
终值 = 512.5M元 / 0.083 = 6,175M元
```

**方法2：退出倍数法（替代方法）**
```
终值 = EBITDA(2029) × 退出倍数

其中：
- 退出倍数 = 当前同行交易中位数（如12-15x EBITDA）

示例：
EBITDA(2029) = 800M元
退出倍数 = 13x

终值 = 800M元 × 13x = 10,400M元
```

**选择一种方法或取两种方法的平均值。**

#### C. 将现金流折现至现值

```
预测 FCF 的现值 = Σ [FCFt / (1 + WACC)^t]，t = 1 至 5

示例：
年份    FCF      折现因子    FCF 现值
        (百万元)                 (百万元)
2025    250元     1/(1.108)^1 = 0.9026    226元
2026    320元     1/(1.108)^2 = 0.8147    261元
2027    390元     1/(1.108)^3 = 0.7353    287元
2028    450元     1/(1.108)^4 = 0.6636    299元
2029    500元     1/(1.108)^5 = 0.5988    299元
                              现值合计：  1,372M元

终值现值 = 终值 / (1 + WACC)^5
终值现值 = 6,175M元 / (1.108)^5 = 6,175M元 × 0.5988 = 3,697M元

企业价值 = 1,372M元 + 3,697M元 = 5,069M元
```

#### D. 计算权益价值和每股价值

```
企业价值                    5,069M元
- 净债务（债务 - 现金）    (450元M)
+ 非经营性资产              0M元
- 少数股东权益              0M元
- 优先股                    0M元
= 权益价值                  4,619M元

稀释流通股数                100M

每股价值 = 4,619M元 / 100M = 46.19元

当前股价：42.00元
隐含上涨空间：10.0%
```

### 步骤3：可比公司分析

#### A. 选择可比公司

**选择标准：**
- 相同行业/板块（首要要求）
- 相似商业模式
- 可比规模（市值、收入）
- 相似增长特征
- 相似地理布局

**识别5-10家同行公司：**
1. [同行1] - 直接竞争对手
2. [同行2] - 直接竞争对手
3. [同行3] - 相邻领域参与者
4. [同行4] - 相似商业模式
5. [同行5] - 区域竞争对手
6. [再添加3-5家]

**记录每家同行的选择依据。**

#### B. 收集同行财务数据

**对每家可比公司，收集：**
- 当前股价
- 流通股数（稀释）
- 市值
- 总债务和现金（用于计算 EV）
- 企业价值
- LTM（过去12个月）财务数据：
  - 收入
  - EBITDA
  - EBIT
  - 净利润
- NTM（未来12个月）一致预期
- 收入增长率
- EBITDA 利润率

**数据来源：**
- Wind / iFinD、Choice / Wind、Wind / iFinD（首选）
- 公司年报/季报获取实际数据
- 东方财富 / 同花顺、雪球 / 东方财富股吧 的一致预期（如无专业工具）

#### C. 计算估值倍数

**对每家同行，计算：**
```
EV/收入 (LTM) = 企业价值 / LTM 收入
EV/收入 (NTM) = 企业价值 / NTM 收入（预估）
EV/EBITDA (LTM) = 企业价值 / LTM EBITDA
EV/EBITDA (NTM) = 企业价值 / NTM EBITDA（预估）
P/E (NTM) = 市值 / NTM 净利润（预估）
```

#### E. 将倍数应用于标的公司

**选择主要倍数（成熟公司通常使用 EV/EBITDA）：**

```
标的公司 NTM EBITDA = 550M元（来自财务模型）

应用同行中位数倍数：
同行中位数 EV/EBITDA (NTM) = 13.8x
隐含 EV = 550M元 × 13.8x = 7,590M元

应用25分位数（保守）：
25分位数 EV/EBITDA (NTM) = 12.5x
隐含 EV = 550M元 × 12.5x = 6,875M元

应用75分位数（乐观）：
75分位数 EV/EBITDA (NTM) = 14.1x
隐含 EV = 550M元 × 14.1x = 7,755M元

估值区间（可比公司法）：6,875M元 - 7,755M元
中点：7,315M元

转换为权益价值：
隐含 EV（中位数）        7,590M元
- 净债务                 (450元M)
= 隐含权益价值           7,140M元

流通股数                 100M
隐含每股价格             71.40元
```

**溢价/折价的理由：**
- 标的增长17% vs. 同行中位数17% → 持平
- 标的 EBITDA 利润率23% vs. 同行中位数23% → 持平
- 标的市场地位 → [论证溢价/折价]
- **结论**：应用中位数倍数（不调整）

#### B. 确定权重依据

**典型权重分配：**
- DCF：40-60%（预测可靠性高时权重更高）
- 交易可比：25-40%（反映市场情绪）
- 可比交易：10-25%（除非并购可能性高，否则权重较低）

**本例：**
- DCF 50%：对预测高度信心
- 可比公司 40%：同行样本充分
- 可比交易 10%：近期并购可能性低

## 合理性检验

**始终执行以下验证检查：**

1. **历史倍数检查**
   - 隐含倍数是否与公司历史交易区间一致？
   - 如不一致，说明原因

2. **同行比较**
   - 相对同行的溢价/折价是否有基本面支撑？
   - 检查：增长、利润率、市场地位

3. **隐含增长检查**
   - 当前价格隐含了怎样的增长预期？
   - 考虑到公司发展轨迹，这是否合理？

4. **市值合理性**
   - 总市值相对于公司规模和同行是否合理？
   - 公司相对于行业是否过大/过小？

5. **终值检查**
   - 终值是否 < 总企业价值的60-70%？
   - 如 > 70%，预测期可能不够长

6. **WACC 合理性**
   - 典型公司 WACC 是否在8-14%区间？
   - 科技/高增长：10-14%
   - 成熟/稳定：7-10%

7. **隐含回报检查**
   - 从当前价格到目标价的12个月 IRR 是多少？
   - 是否与建议评级一致？
