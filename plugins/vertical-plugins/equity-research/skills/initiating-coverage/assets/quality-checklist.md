# 首次覆盖报告质量控制清单

交付首次覆盖报告前，请验证以下所有项目均已完整。

## 关键最低要求 - 报告必须满足

**如以下情况，切勿交付：**
- ❌ DOCX报告少于30页 → 不完整
- ❌ 嵌入图表少于25张 → 不完整
- ❌ 综合性表格少于12个 → 不完整
- ❌ 字数少于10,000字 → 不完整
- ❌ 无XLS财务模型 → 缺少交付物
- ❌ 图表是文字描述而非实际PNG/JPG文件 → 重大失败

## 交付物清单

- [ ] DOCX报告文件已创建
- [ ] XLS财务模型文件已创建
- [ ] 两个文件命名正确：`[公司]_Initiation_Report_[日期].docx` 和 `[公司]_Financial_Model_[日期].xlsx`

## DOCX报告 - 长度与内容

**长度验证：**
- [ ] 报告为30-50页（在最终文档中计算页数）
- [ ] 字数为10,000-15,000字
- [ ] 如少于30页：停止并增加内容

**视觉元素：**
- [ ] 25-35张图表已嵌入（计算图表数量：_____张）
- [ ] 所有图表均为实际PNG/JPG图片文件（非文字描述）
- [ ] 12-20个综合性表格（计算表格数量：_____个）
- [ ] 图表和表格穿插在全文中，非集中在末尾

**图表要求：**
- [ ] 按产品收入图表：堆叠面积图格式 ✓
- [ ] 按地域收入图表：堆叠柱状图格式 ✓
- [ ] DCF敏感性：双向热力图，含颜色编码 ✓
- [ ] 估值区间图：水平柱状图 ✓
- [ ] 所有其他图表均为实际图片文件 ✓

**表格要求：**
- [ ] 完整利润表（40-50行），含5年历史 + 5年预测
- [ ] 完整现金流量表（30-40行）
- [ ] 完整资产负债表（35-45行）
- [ ] 按产品收入表（20-30行）
- [ ] 按地域收入表（15-20行）
- [ ] 按渠道收入表（10-15行）
- [ ] 可比公司表含统计摘要（最大值/75分位/中位数/25分位/最小值）
- [ ] DCF计算表（30-40行）
- [ ] WACC计算表（8-10行）
- [ ] 两个敏感性表
- [ ] 2-3个补充财务/竞争表格

## DOCX报告 - 结构

**第1页要求：**
- [ ] "INITIATING COVERAGE"标题存在（非"Company Update"）
- [ ] 论点导向标题（非事件驱动如"强劲Q4业绩"）
- [ ] 评级框含评级、股价、目标价、52周区间、市值、EV
- [ ] 3-4个段落级要点，使用 ■ 字符和粗体标题
- [ ] 财务与估值指标表，含2-3年历史、2年预测
- [ ] 表格使用"A"后缀表示实际、"E"后缀表示预估
- [ ] 所有视觉元素均有来源行

**内容章节：**
- [ ] 目录（第2页）
- [ ] 投资论点与风险（3-5页）
- [ ] 公司概览（6-12页），包括：
  - [ ] 公司描述
  - [ ] 历史与里程碑
  - [ ] 管理层简介（每位300-400字，3-4位高管）
  - [ ] 产品/服务详情
  - [ ] 竞争格局
- [ ] 财务分析与预测（10-15页）
- [ ] 估值分析（8-12页）
- [ ] 假设章节（2,000-3,000字，记录所有预测假设）
- [ ] 情景分析（1,500-2,000字，含乐观/基准/悲观参数）
- [ ] 附录，含数据来源与参考文献页

## DOCX报告 - 格式

**图表与表格格式：**
- [ ] 每个图表上方有标题："Figure X - [公司] [描述性标题]"
- [ ] 每个图表下方有来源行："Source: [具体来源及日期]"
- [ ] 顺序编号（Figure 1, 2, 3...无遗漏）
- [ ] 每个表格标题行有底色
- [ ] 每个表格底部有来源行
- [ ] 所有年份使用"A"表示实际、"E"表示预估

**专业格式：**
- [ ] 全文一致字体（Calibri、Arial或类似）
- [ ] 页眉页脚含页码
- [ ] 密集排版：60-80%页面覆盖，最少空白
- [ ] 每页同时包含文字和视觉元素（图表或表格）
- [ ] 使用专业商务报告模板

## 引用与来源 ⭐⭐⭐ 关键

**来源归属：**
- [ ] 每个图表有具体来源，含文档名和日期
- [ ] 每个表格有具体来源，含文档引用
- [ ] 正文关键统计数字有脚注含来源
- [ ] 不是泛泛的"公司数据"——必须具体

**超链接：** ⭐⭐⭐ 必须
- [ ] 所有URL均为可点击超链接（非纯文本）
- [ ] 上市公司公告文件超链接至公告查看器
- [ ] 财报记录超链接（雪球或公司IR）
- [ ] 新闻稿超链接至公司IR页面
- [ ] 演示文稿超链接至PDF URL
- [ ] 行业报告超链接（如公开可得）
- [ ] 订阅数据（Wind、iFinD）注明"（需订阅）"
- [ ] 任何地方不显示原始URL——全部格式化为超链接
- [ ] 测试3-5个样本超链接确保可用（Ctrl+Click）

**参考文献页：**
- [ ] 报告末尾有"数据来源与参考文献"页
- [ ] 列出报告中使用的所有来源
- [ ] 按类别组织来源（上市公司公告文件、财报记录等）
- [ ] 每个来源有日期
- [ ] 每个来源有可点击超链接（如适用）

## XLS财务模型 - 结构

**文件结构：**
- [ ] Excel工作簿含15+个标签页
- [ ] 标签页包括：执行摘要、假设、历史财务、收入模型、营业费用、利润表、资产负债表、现金流量、辅助明细、DCF估值、可比公司分析、可比交易、情景、敏感性分析、图表

**格式：**
- [ ] 蓝色文字表示硬编码输入
- [ ] 黑色文字表示公式
- [ ] 绿色文字表示其他标签页链接
- [ ] 专业格式含边框和底色
- [ ] 清晰章节标题和标签

**模型功能：**
- [ ] 所有数字联动（更改假设 → 整个模型更新）
- [ ] DCF链接至假设和预测
- [ ] 无循环引用或错误
- [ ] 所有重要单元格/区域已命名
- [ ] 敏感性表动态运作

## XLS财务模型 - 内容

**预测：**
- [ ] 3-5年历史数据
- [ ] 5年前瞻预测（FY+1至FY+5）
- [ ] 收入按产品、地域、渠道细分
- [ ] 完整利润表含40-50行项目
- [ ] 完整现金流量表含30-40行项目
- [ ] 完整资产负债表含35-45行项目

**估值：**
- [ ] 完整DCF模型含所有计算
- [ ] WACC计算含所有组成部分
- [ ] 终值计算
- [ ] 可比公司分析（5-10家公司）
- [ ] 可比交易分析（5-10笔交易）
- [ ] 情景分析（乐观/基准/悲观）
- [ ] 两个敏感性表
