# 行业概览

## 工作流程

### 第一步：定义范围

- **行业/子行业**：什么行业，定义范围多窄？
- **目的**：客户报告、内部研究、推介材料、创意发掘
- **深度**：高层概览（5-10页）或深度研究（20-30页）
- **角度**：中性概览 vs 主题论点（例如，"AI基础设施建设"）
- **范围**：仅上市公司，还是包含非上市？

### 第二步：市场概览

**市场规模与增长**
- 总可及市场（TAM）及来源
- 历史增长率（5年CAGR）
- 预测增长率和关键假设
- 市场细分（按产品、地域、终端市场、客户类型）

**产业结构**
- 分散 vs 集中——前5名市场份额
- 价值链图——价值在哪里积累？
- 商业模式类型（订阅、交易、许可、服务）
- 进入壁垒（资本、监管、技术、网络效应）

**关键趋势与驱动因素**
- 长期顺风因素（3-5个主要趋势）
- 逆风因素和风险
- 技术颠覆方向
- 监管动态
- 并购活动和整合趋势

### 第三步：竞争格局

**公司概况**（前5-10名参与者）：

| 公司 | 收入 | 增长 | EBITDA利润率 | 市场份额 | 关键差异化因素 |
|------|------|------|-------------|---------|-------------|
| | | | | | |

每家公司简要概况：
- 业务描述（2-3句话）
- 战略定位和护城河
- 近期动态（财报、并购、产品发布）
- 估值快照（P/E, EV/EBITDA, EV/Revenue）

**竞争动态**
- 公司如何竞争？（价格、产品、服务、分销）
- 谁在获取/失去份额，为什么？
- 新进入者或相邻参与者的颠覆风险

### 第四步：估值背景

- 行业交易倍数（当前和历史区间）
- 溢价/折价驱动因素（增长、利润率、市场地位）
- 近期并购交易倍数
- 行业相对整体市场的表现

### 第五步：投资启示

- 最佳风险/回报机会在哪里？
- 通过该行业可以表达什么主题押注？
- 行业关键争论（多头 vs 空头论点）
- 可能改变行业叙事的催化剂

### 第六步：输出

- Word文档或PowerPoint，包含：
  - 市场概览和规模
  - 竞争格局图
  - 公司比较表
  - 估值摘要
  - 关键图表：市场增长、份额趋势、估值历史
- Excel附录，含详细公司数据

## 重要提示

- 所有市场规模数据必须注明来源——引用研究机构或方法论
- 区分TAM炒作和现实可及市场
- 行业概览更新很快——注明日期并标记可能过时的数据
- 图表必不可少——市场规模瀑布图、竞争定位矩阵、估值散点图
- 如果面向客户，将"so what"定制到他们的具体情况（并购目标识别、竞争定位、市场进入）
