# 财报预览

## 工作流程

### 第一步：收集背景信息

- 确认公司和报告季度
- 通过网络搜索获取一致预期（收入、EPS、关键分部指标）
- 查找财报日期和时间（盘前 vs 盘后）
- 回顾公司上一季度的电话会议记录，了解任何指引或评论

### 第二步：关键指标框架

建立针对特定公司的"关注要点"框架：

**财务指标：**
- 收入 vs 一致预期（总计及按分部）
- EPS vs 一致预期
- 利润率（毛利率、营业利润率、净利润率）——扩张还是收缩？
- 自由现金流
- 前瞻指引 vs 一致预期

**运营指标**（按行业）：
- 科技/SaaS：ARR、净留存率、RPO、客户数量
- 零售：同店销售、客流量、客单价
- 工业：订单积压、订单出货比、价格 vs 销量
- 金融：净息差（NIM）、信贷质量、贷款增长、手续费收入
- 医疗：处方量、患者量、管线更新

### 第三步：情景分析

构建3种情景及股价影响：

| 情景 | 收入 | EPS | 关键驱动因素 | 股价反应 |
|------|------|-----|------------|---------|
| 乐观 | | | | |
| 基准 | | | | |
| 悲观 | | | | |

对每种情景：
- 运营层面需要发生什么
- 管理层什么表态会暗示这种情景
- 历史背景——在类似业绩公布时股价如何变动

### 第四步：催化剂清单

识别将决定股价反应的3-5个因素：

1. [指标] vs [一致预期/耳语数]——为什么重要
2. [指引项目]——买方期望听到什么
3. [叙事变化]——任何战略变化、并购、重组

### 第五步：输出

一页纸财报预览，包含：
- 公司、季度、财报日期
- 一致预期表
- 关键关注指标（按重要性排序）
- 乐观/基准/悲观情景表
- 催化剂清单
- 交易设置：近期股价表现、期权隐含波动

## 重要提示

- 一致预期会变化——始终注明来源和日期
- 买方调查的"耳语数"通常比公布的共识更相关
- 历史财报反应有助于校准预期（搜索"[公司] earnings reaction history"）
- 期权隐含波动告诉你市场预期——与你的情景进行比较
