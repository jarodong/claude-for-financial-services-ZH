# 盈利更新详细工作流程

本文档提供盈利更新流程每个阶段的详细逐步说明。

## ⚠️⚠️⚠️ 关键警告：务必使用最新盈利数据 ⚠️⚠️⚠️

**请先阅读此部分：**

训练数据已过时。务必主动搜索并获取最新的盈利材料。使用过时盈利数据是盈利分析中的头号错误。

**开始前：**
1. **确认今日日期** —— 写下当前日期
2. **搜索最新数据** —— 使用网络搜索查找最新盈利
3. **验证日期** —— 确认盈利公告在最近3个月内
4. **如超过3个月** —— 获取了错误季度，重新搜索

## 阶段1：盈利数据收集（30-60分钟）

### 步骤1：确定最新盈利周期

**关键**：务必搜索最新盈利——不要依赖知识截止日期。
**关键**：绝不使用训练数据中的盈利数据——它已过时。

**步骤1a：搜索最新盈利公告**

**🚨 主动搜索——训练数据已过时。🚨**

**必做步骤1：确认今日日期**
- **明确写下今日日期**：[月] [日]，[年]
- **用此验证**任何找到的盈利数据在3个月以内
- **示例**："今天是2024年10月29日"

**必做步骤2：搜索"最新盈利"**
- **使用网络搜索**，查询如：
  - `[公司名称] latest earnings results`
  - `[公司名称] most recent quarterly earnings`
  - `[股票代码] earnings latest quarter`
- **或搜索公司投资者关系网站**：
  - 访问 `investor.[公司].com` 或 `[公司].com/investors`
  - 导航至"Press Releases"、"News"或"Earnings"部分
  - **按日期排序查找最新公告**
  - 查找关键词："earnings"、"results"、"financial results"、"quarterly results"

**必做步骤3：验证公告日期**
- **查看找到的盈利公告日期**
- **计算**：该日期是否在今日之前3个月内？
- **如果是** → 进入下一步
- **如果否（超过3个月）** → 🚨 错误季度——重新搜索更近期的

**❌ 需避免的常见错误：**
- 未搜索就使用训练数据中的盈利数据
- 基于预期假设"Q3 2024"是最新的
- 未核查日期就取用第一个找到的盈利公告
- 未将公告日期与今日日期对比
- 公告已4个月以上仍继续推进

**✅ 正确做法：**
- 先确认今日日期
- 明确搜索"latest"或"most recent"
- 阅读材料上的实际公告日期
- 确认公告日期在今日3个月以内
- 如有疑虑，用不同关键词重新搜索

**必做步骤4：识别季度**
- **阅读标题/头条**识别季度（Q1、Q2、Q3、Q4或财季）
- **阅读文件上的公告日期**
- **验证季度名称和日期都是近期的**

3. **如投资者关系网站不清晰，替代搜索方法：**
   - 网络搜索：`[公司名称] latest earnings results`
   - 网络搜索：`[公司名称] most recent quarterly earnings`
   - 网络搜索：`[股票代码] earnings latest quarter`
   - 巨潮资讯网：搜索公司，查看最近季报或年报申报日期

**能查到最新数据的搜索示例：**
- "贵州茅台 latest earnings results" → 返回最近已报告季度
- "宁德时代 most recent quarterly earnings" → 显示最新宁德时代盈利
- "比亚迪 Q3 2024 earnings" → 确认Q3 2024存在

**步骤1b：了解公司财年日历**

通过搜索确定最新季度后，了解公司财年以正确解读：

**常见财年模式：**
- **日历年（CY）**：Q1=1-3月，Q2=4-6月，Q3=7-9月，Q4=10-12月
- **李宁财年**：Q1=6-8月，Q2=9-11月，Q3=12-2月，Q4=3-5月（5月财年结束）
- **比亚迪财年**：Q1=1-3月，Q2=4-6月，Q3=7-9月，Q4=10-12月（12月财年结束）
- **永辉超市财年**：Q1=1-3月，Q2=4-6月，Q3=7-9月，Q4=10-12月（12月财年结束）

许多公司在盈利公告标题中注明其财年。如需要，搜索`[公司] fiscal year calendar`。

**步骤1c：强制验证——确认获取的是最新数据**

🛑 **暂停——在验证以下所有项之前不要继续：**

- [ ] 已写下今日日期：[月] [日]，[年]
- [ ] 已使用"latest earnings"或"most recent earnings"主动搜索
- [ ] 已找到盈利公告日期：[月] [日]，[年]
- [ ] 已验证公告在今日3个月以内（实际计算！）
- [ ] 未仅基于今日日期假设季度
- [ ] 能看到确认该季度/期间的实际新闻稿
- [ ] 已打开并阅读实际盈利材料（而非仅假设存在）

**🚨 危险信号——如以下任一为真，则获取了错误季度：**
- 公告日期已超过90天
- 依赖预期而非搜索结果
- 实际上未看到确认该季度存在的新闻稿或申报文件
- 未搜索就使用了训练数据中的数据
- 无法说明确切公告日期
- 找到的公告日期为2023年或更早（而今天是2024年或以后）

**如有任何危险信号**：暂停并重新搜索。不要使用过时数据继续。

**步骤1c：处理命名差异**

公司使用不同术语——识别以下模式：

**季度术语：**
- "Q1 2024"、"Q1 FY24"、"First Quarter 2024"、"1Q24"
- "Third Quarter Fiscal 2024"、"Q3 FY2024"、"3Q FY24"

**盈利公告标题：**
- "[Company] Reports Q3 2024 Results"
- "[Company] Announces Third Quarter Fiscal 2024 Financial Results"
- "[Company] Q3 Revenue Grew 15% Year-over-Year"

**上市公司公告搜索：**
- 公司名称可能与常用名称不同（如"Meta Platforms, Inc." vs "Facebook"）
- 用股票代码搜索可更可靠地找到申报文件
- 查找最近的季报（季度）或年报（如为Q4则为年度）

### 步骤2：收集盈利材料

搜索并确认最新季度后，收集以下材料：

**⚠️ 重要：搜索并获取实际文件——不要依赖训练数据。**

**主要材料（必需）：**
- **盈利新闻稿** —— 通常在公司投资者关系网站"Press Releases"或"News"栏目下
  - 导航至IR网站，找到实际新闻稿
  - 搜索模式："[公司名称] latest earnings"、"[公司名称] Q[X] [年份] earnings results"
  - 查找PDF或HTML版本
  - **验证日期与步骤1找到的一致**（应在最近1-3个月内）
  - **阅读实际文件**确认季度并获取报告数字

- **季报或年报申报** —— 在巨潮资讯网（cninfo.com.cn）
  - 用股票代码搜索
  - 第1-3季度：查找最近的季报
  - Q4：查找年报（年度报告）
  - 注：可能在盈利公告后1-5天才申报
  - 直接链接格式：`https://www.cninfo.com.cn/new/disclosure/detail?stockCode=[stock-code]`

- **电话会议纪要** —— 🚨 **验证纪要上的日期** 🚨
  - **搜索**："[Company] latest earnings call transcript"或"[Company] Q[X] [年份] earnings call transcript"
  - **来源**：
    - 公司IR网站（部分直接发布纪要）
    - 雪球 / 东方财富股吧：搜索"[公司名称] [最新季度] 电话会议纪要"
    - AlphaStreet、Motley Fool（替代来源）
  - **关键日期检查**：
    - 使用任何纪要前，验证纪要本身的日期
    - 纪要日期必须与步骤1的盈利公告日期匹配
    - 如纪要显示"Q2 2023"但公告为"Q3 2024"，则获取了错误纪要
    - 常见错误：未核查日期就取用旧纪要
  - 如纪要尚未可用，可听网络重播或注明等待纪要

**补充材料（如有）：**
- **投资者演示/幻灯片** —— 通常与新闻稿一起发布在IR网站
  - 通常标题为"Q[X] [年份] Earnings Presentation"或"Investor Presentation"
  - PDF格式，含管理层在电话会议中演示的幻灯片

- **补充数据文件** —— 部分公司提供含详细指标的Excel文件
  - 查找"Supplemental Financial Information"或"Investor Data Sheet"

**参考材料（用于比较）：**
- **上季度业绩** —— 用于环比比较
  - 来自上季度盈利公告（90天前）

- **去年同期同季** —— 用于同比比较
  - 来自去年同季（4个季度前）

- **前期预测** —— 如此前已覆盖该公司
  - 来自上次盈利更新或首次覆盖报告
  - 检查本季度指标的预测值

- **一致预期** —— 来自Wind / iFinD或东方财富 / 同花顺
  - 关键：使用盈利公告前的预期
  - 查找"as of [盈利前日期]"确保为公告前一致预期
  - 超预期/低于预期分析所需

**🛑 进入步骤3前的强制验证：**

**日期——验证所有日期一致：**
- [ ] 已写下今日日期：_______________
- [ ] 盈利公告日期：_______________（必须在今日3个月以内）
- [ ] 电话会议纪要日期：_______________（必须与公告日期±1天一致）
- [ ] 季报/年报申报日期：_______________（必须与公告为同一季度）
- [ ] 所有材料显示同一季度（如全部显示"Q3 2024"，非混合季度）

**搜索与获取——验证主动搜索已完成：**
- [ ] 已搜索"latest earnings"（非基于当前日期假设）
- [ ] 已获取并阅读实际盈利新闻稿
- [ ] 已打开实际电话会议纪要并验证日期
- [ ] 已通过日期核实确认这是最近季度
- [ ] 从实际公告获取了完整财务业绩（营收、每股收益、利润率等）
- [ ] 有盈利前一致预期及来源日期

**🚨 危险信号——如以下任一为真，暂停：**
- 实际上未搜索或获取盈利材料
- 凭记忆或训练数据而非当前文件工作
- 盈利公告日期已超过90天
- 无法说明盈利公告的确切日期
- 纪要日期与公告日期不一致
- 材料显示不同季度（如公告为Q3但纪要为Q2）
- 未验证日期就取用了第一个结果

### 步骤4：识别电话会议关键主题

收听或阅读电话会议纪要，记录：
- 管理层语气（自信、谨慎、防御？）
- 强调的关键话题（产品发布、地区趋势、竞争）
- 分析师提问（投资者关注什么？）
- 提供的指引（上调、下调、维持、首次给出？）
- 任何意外或出乎意料的评论

## 阶段2：分析（2-3小时）

### 步骤5：超预期/低于预期分析

对每个超预期或低于预期的关键指标，解释：

**如超预期：**
- 什么推动了超预期表现？
- 是一次性还是可持续？
- 管理层是否上调了前瞻指引？
- 这如何影响我们的投资论点？

**如低于预期：**
- 问题出在哪里？
- 是公司特有还是行业普遍？
- 管理层是否采取纠正措施？
- 这如何影响我们的投资论点？

**示例格式：**
```
■ **营收超预期3%，得益于DTC业务的强劲表现**

营收135亿元超出我们131亿元的预测4亿元（3%），
超出一致预期132亿元3亿元（2%）。超预期主要由直接面向消费者
渠道推动，该渠道同比增长18%（我们预测为12%），抵消了弱于预期的
批发业务（-5%，我们预测持平）。管理层将强劲的数字需求和成功的产品
发布（飞电4跑鞋、新国潮联名配色）列为关键驱动因素。DTC目前占
总营收的42%，去年同期为38%，验证了渠道转型策略的成功。
```

### 步骤6：分业务/地区/产品分析

按以下维度分析表现：
- 业务分部（如为多分部公司）
- 地区（北美、欧洲、中国等）
- 产品类别
- 渠道（零售、批发、电商）

识别：
- 什么超出预期？
- 什么低于预期？
- 与前几个季度的趋势对比
- 管理层对各领域前景的评论

### 步骤7：利润率分析

分析盈利能力：
- 毛利率：上升还是下降？原因？
- 营业利润率：上升还是下降？原因？
- 关键驱动因素（定价、组合、成本、杠杆）
- 前瞻展望

### 步骤8：指引分析

如公司提供了指引：
- 将新指引与前期指引对比
- 与内部预测和市场一致预期对比
- 评估可信度（公司是否有保守指引的习惯？超预期记录？）
- 识别指引背后的关键假设

如公司未提供指引：
- 明确说明
- 根据业绩和评论提供独立展望

### 步骤9：更新财务模型

更新以下预测：
- 当年（剩余季度）
- 次年
- 可能包括第三年

**清晰展示：**
```
更新预测：
─────────────────────────────────────────────────
                    旧预测      新预测      变动       原因
FY2024E 营收        XX元.XB      XX元.XB      +X.X%     [简要原因]
FY2024E EBITDA      X.XB元       X.XB元       +X.X%     [简要原因]
FY2024E 每股收益     X.XX元       X.XX元       +X.X%     [简要原因]

FY2025E 营收        XX元.XB      XX元.XB      +X.X%     [简要原因]
FY2025E EBITDA      X.XB元       X.XB元       +X.X%     [简要原因]
FY2025E 每股收益     X.XX元       X.XX元       +X.X%     [简要原因]
```

### 步骤10：更新估值与目标价

基于更新预测：
- 重新计算DCF（使用更新现金流）
- 更新可比公司倍数（如已发布同行财报）
- 确定新公允价值
- 决定是否调整目标价

**目标价决策：**
- 如预测变动显著（>5%） → 通常调整目标价
- 如预测变动较小（<5%） → 可能维持目标价
- 如论点增强/削弱 → 即使预测未变也可能调整

### 步骤11：评估评级影响

决定是否调整评级：
- 如业绩显著超预期 + 指引上调 → 考虑上调评级
- 如业绩显著低于预期 + 指引下调 → 考虑下调评级
- 如符合预期或好坏参半 → 通常维持评级

**考虑因素：**
- 股价反应（上涨/下跌/持平？）
- 估值（相对新预测偏贵还是偏便宜？）
- 风险/收益（非对称性是否改变？）

## 阶段3：图表生成（1-2小时）

### 步骤12：生成8-12张图表

创建聚焦季度趋势和新变化的图表。

**必需图表（共8-12张）：**

1. **季度营收走势**（柱状图）
   - 最近8-12个季度
   - 展示每季度超预期/低于预期 vs. 预测
   - 突出当前季度

2. **季度每股收益走势**（柱状图）
   - 最近8-12个季度
   - 展示超预期/低于预期 vs. 预测
   - 扣非后和企业会计准则口径

3. **季度利润率趋势**（折线图）
   - 毛利率、EBIT利润率、净利率
   - 最近8-12个季度
   - 展示走势

4. **分业务/地区营收**（堆叠柱状图或表格）
   - 当季 vs. 同比
   - 分业务增长率

5. **关键经营指标**（多线图）
   - 客户数、ARPU、销量等（相关指标）
   - 最近8-12个季度

6. **超预期/低于预期摘要**（瀑布图或表格）
   - 展示超预期/低于预期的构成
   - 什么驱动了与预测的差异

7. **预测修正图**（前后比较）
   - 旧FY预测 vs. 新FY预测
   - 展示变动的柱状图

8. **估值图**（市盈率或EV/EBITDA倍数）
   - 历史倍数区间
   - 当前倍数
   - 公允价值倍数

**可选图表（如空间允许）：**
- 同行比较（如已发布同行财报）
- 指引 vs. 市场一致预期比较
- 现金流指标
- 资产负债表要点（如值得关注）

**图表样式指南：**
- 聚焦趋势（季度走势）
- 突出变化（超预期/低于预期、预测修正）
- 保持简洁清晰（这是快速出稿报告）

## 阶段4：报告撰写（2-3小时）

### 步骤13：创建DOCX报告

使用DOCX技能创建8-12页报告。

详见 [report-structure.md](report-structure.md) 了解完整逐页模板和格式要求。

**关键步骤：**
1. 创建第1页，含盈利摘要和快速要点
2. 添加详细业绩分析（第2-3页）
3. 包含关键指标和指引（第4-5页）
4. 更新投资论点（第6-7页）
5. 提供估值和预测（第8-10页）
6. 如需要添加附录（第11-12页）
7. 全文嵌入8-12张图表
8. 添加1-3个摘要表格
9. 包含完整来源部分，附可点击超链接

### 步骤14：可选——更新XLS模型
