# 最佳实践、示例与质量指南

本文档提供示例、成功技巧、常见错误规避和综合质量清单。

## 标题示例

### 优质盈利更新标题：
- "李宁 Q2 FY24: DTC业务强劲抵消批发疲弱 - 维持增持，目标价95元"
- "宁德时代 Q3'24: 产能扩张超预期 - 上调预测，目标价285元"
- "LVMH Q4'24: Fashion & Leather Resilient, Wines Weak - In-Line, Reiterating Buy"
- "比亚迪 Q1 FY24: 新能源车销量超预期，电池业务低于预期 - 季度表现分化，下调目标价至185元"

### 差标题（避免使用）：
- "李宁季度更新"（过于笼统，无观点）
- "Company Reports Earnings"（陈述显而易见的事实，无分析）
- "Q3 Results Analysis"（无公司名称，无观点）

## 成功技巧

1. **速度至关重要**：在财报后24-48小时内发出，而非数天之后

2. **结论前置**：超预期还是低于预期？上调还是下调预测？

3. **量化一切**："强劲"毫无意义，"在12亿元营收基础上超预期1.5亿元"才清晰

4. **聚焦驱动因素**：不要只说"营收超预期"，要解释原因

5. **展示推演过程**：旧预测 → 新预测，附变更原因

6. **重大变化需更新目标价**：如预测变动>5%，通常目标价也应调整

7. **回应电话会议要点**：引用管理层评论，不要只分析新闻稿

8. **与同行比较**：如类似公司已发布财报，注意相对表现

9. **保持简洁**：这不是综合报告，聚焦季度业绩

10. **用图表呈现趋势**：季度走势图表最具价值

## 常见错误规避

过于全面：不要为季度业绩写出首次覆盖级别的报告

遗漏超预期/低于预期分析：首先说明业绩是否超预期

未更新预测：必须提供更新的前瞻预测

语言模糊："表现强劲"却无量化支撑

忽视指引：如公司给出指引，需充分分析

速度太慢：财报5天后发出已失去时效性

重复基础知识：不要花3页解释公司是做什么的

遗漏目标价更新：如预测重大变动，目标价也应调整

无投资影响分析：必须将业绩与投资论点和评级关联

缺少引用：每个数字都需要附可点击超链接的来源

纯文本URL：所有URL必须格式化为可点击超链接

## 综合质量控制清单

在交付盈利更新前，验证以下所有项目：

### 内容与分析清单

**超预期/低于预期分析：**
- [ ] 超预期/低于预期分析引领报告
- [ ] 量化具体差异（如"超预期1.2亿元，幅度3%"）
- [ ] 解释业绩与预期差异的原因
- [ ] 分析每个关键指标（营收、每股收益、利润率等）

**指标与业绩：**
- [ ] 所有关键指标均含同比比较分析
- [ ] 相关处包含环比比较
- [ ] 提供分业务/地区/产品拆解
- [ ] 分析经营指标（客户数、ARPU、销量等）

**指引与预测：**
- [ ] 分析并量化指引变化（如有）
- [ ] 如无指引，明确说明
- [ ] 提供当年更新预测
- [ ] 提供次年更新预测
- [ ] 清晰展示新旧预测对比
- [ ] 解释变更内容及原因

**估值与评级：**
- [ ] 根据业绩更新目标价（如合理）
- [ ] 如目标价不变，明确维持
- [ ] 解释估值方法论
- [ ] 确认或调整评级，附明确理由
- [ ] 评估并在需要时更新投资论点

### 格式与篇幅清单

**整体结构：**
- [ ] 报告为8-12页（不短于、不长于此）
- [ ] 第1页采用盈利摘要格式
- [ ] 标题含"EARNINGS UPDATE"（非"Initiating Coverage"）
- [ ] 事件驱动型标题（如"Strong Q3 Results..."）

**表格：**
- [ ] 包含1-3个摘要表格（非综合报表）
- [ ] 所有表格有清晰列标题
- [ ] 所有表格标题行有底色
- [ ] 所有表格底部有来源行
- [ ] 预测表格展示新旧对比及变动列

**图表：**
- [ ] 全文嵌入8-12张图表
- [ ] 所有图表上方有"Figure X - [标题]"说明
- [ ] 所有图表下方有"Source: [来源]"行
- [ ] 图表聚焦季度趋势
- [ ] 图表突出变化（超预期/低于预期、预测修正）
- [ ] 图表采用专业样式

### 引用与来源清单 ⭐⭐⭐ 强制要求

**图表与表格引用：**
- [ ] 每个图表均标注具体文件名称和日期的来源
- [ ] 每个表格均标注文件引用来源
- [ ] 来源引用包含适用的页码或幻灯片编号

**超预期/低于预期引用：**
- [ ] 超预期/低于预期分析引用一致预期来源（Wind / iFinD等）
- [ ] 一致预期来源包含"as of"日期（财报前收盘）
- [ ] 公司报告业绩引用盈利公告或季报

**指引引用：**
- [ ] 当前指引引用电话会议纪要或公告
- [ ] 前期指引引用上季度材料
- [ ] 当前和前期指引来源均附超链接

**统计数据与指标：**
- [ ] 关键统计数据附带来源的脚注
- [ ] 脚注引用具体文件和页码/幻灯片编号
- [ ] 管理层引言标注发言人姓名和来源文件

**超链接：** ⭐⭐⭐ 关键要求
- [ ] 所有URL均为可点击超链接（非纯文本）
- [ ] 超链接格式化为有意义的显示文本
- [ ] Word文档中为蓝色、带下划线的超链接格式
- [ ] 超链接已测试且可用（Ctrl+Click可打开正确页面）
- [ ] 所有上市公司公告文件均链接至公告查看器
- [ ] 所有盈利材料均附超链接（公告、纪要、演示）
- [ ] 上季度材料附超链接用于比较
- [ ] 文档中任何位置无纯文本URL

**来源部分：**
- [ ] 报告末尾包含"Sources & References"部分
- [ ] 该部分列出所有盈利材料及日期
- [ ] 所有材料附可点击超链接
- [ ] 列出一致数据来源（订阅数据即使无链接也需列出）
- [ ] 包含前期参考

### 准确性清单

**数值准确性：**
- [ ] 数字与公司报告业绩完全一致
- [ ] 所有计算数学验证无误
- [ ] 预测变动计算正确
- [ ] 估值数学准确
- [ ] 图表与文字描述一致

**事实准确性：**
- [ ] 股票代码无拼写错误
- [ ] 公司名称无拼写错误
- [ ] 日期当前且准确
- [ ] 季度/年份引用正确
- [ ] 年份标注正确（A为实际值，E为预测值）

### 时效性清单

**发布时间：**
- [ ] 报告在财报发布后24-48小时内发出
- [ ] 如超过48小时，注明为"延迟反应"
- [ ] 已通过搜索最新财报验证所有数据来自最新季度
- [ ] 未依赖知识截止日期——主动搜索了当前数据
- [ ] 一致预期为财报前数据（非财报后）
- [ ] 无过时信息
- [ ] 盈利公告日期在最近1-3个月内（非6个月以上）

### 写作风格清单

**清晰与直接：**
- [ ] 数字引领表述（"营收增长15%至12亿元"而非"营收强劲"）
- [ ] 使用"vs."而非"versus"
- [ ] 全文直接简洁
- [ ] 聚焦新信息（不重复公司基础知识）
- [ ] 避免模糊语言（"表现强劲"却无量化）

**专业标准：**
- [ ] 保持机构化语气
- [ ] 术语使用一致
- [ ] 无非正式语言
- [ ] 使用规范的财务标注

## 交付前最终检查

在发送报告给用户前，快速完成以下最终检查：

### 5分钟最终审查：
1. **第1页**：评级是否清晰？目标价是否更新？关键要点是否有力？
2. **数字**：报告业绩是否与公司新闻稿完全一致？
3. **引用**：抽查3-4个图表/表格——所有来源均附可点击超链接？
4. **预测**：新旧对比是否清晰？变动是否解释？
5. **图表**：8-12张全部嵌入？全部编号并附说明？
6. **篇幅**：是否为8-12页（不是6页，也不是15页）？
7. **超链接**：测试3-4个超链接——Ctrl+Click是否可用？
8. **时效性**：是否在财报后48小时内发出？

如所有项目均通过检查，报告即可交付。

## 交付摘要格式

向用户交付完成报告时，提供以下摘要：

```
[公司] Q[X] [年份] 盈利更新完成

业绩：[超预期 / 符合预期 / 低于预期]
- 营收：X.XB元（[超/低于]预期XXM元，幅度X%）
- 每股收益：X.XX元（[超/低于]预期X.XX元）

关键要点：
■ [要点1]
■ [要点2]
■ [要点3]

更新预测：
- FY[年份]E 营收：XX.XB元（此前：XX.XB元，[+/-]X%）
- FY[年份]E 每股收益：X.XX元（此前：X.XX元，[+/-]X%）

评级：[维持 / 上调 / 下调] [评级]
目标价：XXX元（此前：XXX元）- [+/-]XX%上涨空间

交付物：
✓ 8-12页盈利更新报告（DOCX）
✓ 8-12张嵌入图表
✓ 更新预测，含新旧对比
✓ 完整来源部分，附可点击超链接
✓ [可选：更新XLS财务模型]

文件：[公司]_Q[X]_[年份]_Earnings_Update.docx
```
