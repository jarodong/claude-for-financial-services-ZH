# 催化剂日历

## 工作流程

### 第一步：定义覆盖范围

- 要跟踪的公司列表（股票代码或名称）
- 行业/板块聚焦
- 是否包含宏观事件？（美联储会议、经济数据、监管截止日期）
- 时间范围（未来2周、1个月、1个季度）

### 第二步：收集催化剂

为每家公司识别即将发生的事件：

**财报与财务事件**
- 季度财报日期和时间（盘前/盘后）
- 年度股东大会
- 投资者日/分析师日
- 资本市场日
- 债务到期/再融资日期

**公司事件**
- 产品发布或公告
- FDA审批/监管决策
- 合同续签或到期
- 并购里程碑（交割日期、监管审批）
- 管理层变动
- 内部交易窗口（锁定期到期）

**行业事件**
- 主要会议（日期、哪些公司出席）
- 行业展会
- 监管意见征询期或裁决
- 行业数据发布（月度销售、流量等）

**宏观事件**
- 美联储会议（FOMC日期）
- 就业报告、CPI、GDP发布
- 央行决策（ECB、BOJ等）
- 具有市场影响的地缘政治事件

### 第三步：日历视图

| 日期 | 事件 | 公司/行业 | 类型 | 影响程度（高/中/低） | 我们的持仓 | 备注 |
|------|------|----------|------|-------------------|-----------|------|
| | | | 财报/公司/行业/宏观 | | 多头/空头/中性 | |

### 第四步：每周预览

每周生成前瞻性摘要：

**本周关键事件：**
1. [日期]：[公司] Q[X] 财报——一致预期 [X美元 EPS]，我们的预估 [X美元]，关键关注点：[指标]
2. [日期]：[事件]——对[股票]的重要性
3. [日期]：[宏观数据发布]——预期和持仓策略

**下周预览：**
- 提前关注即将到来的重要事件

**持仓影响：**
- 可能影响特定持仓的事件
- 是否建议提前布局
- 二元事件前的风险管理

### 第五步：输出

- 带有日历视图和可排序列的Excel工作簿
- 每周预览邮件/备忘录（markdown格式）
- 可选：与Google日历集成

## 重要提示

- 财报日期可能变动——临近日期时需对照公司投资者关系页面和Wind / iFinD进行验证
- 预先公告风险：跟踪有预先公告历史（正面或负面）的公司
- 会议参会名单很有价值——哪些公司将出席，哪些公司明显缺席？
- 某些催化剂是重复性的（月度行业数据）——建立模板并自动填充
- 按影响程度进行颜色编码：红色=高影响，黄色=中等，绿色=常规
- 将过去的催化剂与实际结果一起归档——随着时间积累可培养模式识别能力
