# 模型更新

## 工作流程

### 第一步：识别变化内容

确定更新触发因素：
- **财报发布**：新的季度实际数据需要录入
- **指引变化**：公司更新了前瞻展望
- **预估修正**：分析师基于新数据调整假设
- **宏观更新**：利率、汇率、大宗商品价格变化
- **事件驱动**：并购、重组、新产品、管理层变动

### 第二步：录入新数据

#### 财报后

用已公布的实际数据更新模型：

| 项目 | 之前预估 | 实际值 | 差异 | 备注 |
|------|---------|--------|------|------|
| 收入 | | | | |
| 毛利率 | | | | |
| 营业费用 | | | | |
| EBITDA | | | | |
| EPS | | | | |
| [关键指标1] | | | | |
| [关键指标2] | | | | |

**分部详情**（如适用）：
- 更新每个分部的收入和利润率
- 注意任何分部组合变化

**资产负债表/现金流量表更新**：
- 现金和债务余额
- 股份数量（回购、稀释）
- 资本支出实际 vs 预估
- 营运资金变化

### 第三步：修正前瞻预估

基于新数据，调整前瞻预估：

| | 旧年度预估 | 新年度预估 | 变化 | 旧下年度 | 新下年度 | 变化 |
|---|-----------|-----------|------|---------|---------|------|
| 收入 | | | | | | |
| EBITDA | | | | | | |
| EPS | | | | | | |

**关键假设变化：**
- 你在改变什么假设，为什么？
- 收入增长率：旧 → 新（原因）
- 利润率假设：旧 → 新（原因）
- 任何新项目（重组费用、一次性收益等）

### 第四步：估值影响

用更新后的预估重新计算估值：

| 估值方法 | 之前 | 更新后 | 变化 |
|---------|------|--------|------|
| DCF公允价值 | | | |
| P/E（NTM EPS × 目标倍数） | | | |
| EV/EBITDA（NTM EBITDA × 目标倍数） | | | |
| **目标价** | | | |

### 第五步：摘要与行动

**预估变化摘要：**
- 一段话：什么变了、为什么变、对股票意味着什么
- 这是改变论点的事件还是噪音？

**评级/目标价：**
- 维持还是改变评级？
- 新目标价（如有变化）及方法论
- 相对当前价格的上行/下行空间

### 第六步：输出

- 更新后的Excel模型（如用户提供现有模型）
- 预估变化摘要（markdown或Word）
- 更新后的目标价推导

## 重要提示

- 向前推算前，始终将你的预估与公司报告数据进行核对
- 注意任何非经常性项目，以及你的预估是企业会计准则口径还是扣非后口径
- 跟踪你的预估修正历史——它展示你的分析演进过程
- 如果季度数据噪音大，在预估变化中区分信号和噪音
- 更新后检查一致预期——你修正后的预估与华尔街相比如何？
- 股份数量很重要——股权激励、可转债或回购的稀释可能显著影响EPS
