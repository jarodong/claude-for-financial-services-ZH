# 财报分析命令

创建专业的股票研究财报更新报告，分析季度业绩。

## 工作流程

### 第一步：收集信息

解析输入以获取：
- 公司名称或股票代码
- 季度（如Q3 2024、Q2 FY25）

如未提供，则询问：
- "您想分析哪家公司的财报？"
- "哪个季度？（如Q3 2024）"

### 第二步：验证时效性

**关键**：继续之前，验证您拥有最新数据：
1. 搜索"[公司] latest earnings results [当前年份]"
2. 确认财报发布在最近3个月内
3. 确认记录日期与发布日期匹配

如数据过时，告知用户并搜索最新的。

### 第三步：加载财报分析技能

使用 `skill: "earnings-analysis"` 创建报告：

1. **数据收集**（搜索最新）：
   - 财报发布（新闻稿）
   - 季报文件
   - 财报电话会议记录
   - 投资者演示/补充材料
   - 一致预期（Wind / iFinD）

2. **超预期/不及预期分析**：
   - 收入 vs 一致预期：超预期/不及预期 $X 或 X%
   - EPS vs 一致预期：超预期/不及预期 $X 或 X%
   - 关键分部表现 vs 预期
   - 解释业绩为何不同

3. **关键指标分析**：
   - 按分部/地域收入细分
   - 利润率趋势（毛利率、营业利润率、净利润率）
   - 指引：上调/维持/下调
   - 更新前瞻预估

4. **生成图表**（8-12张）：
   - 季度收入走势
   - 季度EPS走势
   - 利润率趋势
   - 按分部收入
   - 超预期/不及预期摘要
   - 预估修正
   - 估值图表

5. **创建报告**（8-12页）：
   - 第1页：含评级和目标价的摘要
   - 第2-3页：详细业绩分析
   - 第4-5页：关键指标与指引
   - 第6-7页：更新投资论点
   - 第8-10页：估值与预估
   - 来源部分含可点击超链接

### 第四步：交付输出

提供：
1. **DOCX报告** - 8-12页财报更新
2. **摘要**，突出：
   - 关键指标超预期/不及预期
   - 指引变化
   - 论点影响（正面/负面/中性）

## 报告结构参考

```
第1页：财报摘要
┌─────────────────────────────────────────────────────────────────┐
│ [公司] Q3 2024 财报更新                                          │
│ 评级：买入 | 目标价：$XXX（原$XXX）                               │
├─────────────────────────────────────────────────────────────────┤
│ 关键要点                                                         │
│ • 收入超预期X%，受强劲[分部]表现驱动                              │
│ • EPS超预期$X.XX，由利润率扩张驱动                                │
│ • 全年指引上调至$X.XX-$X.XX（原$X.XX-$X.XX）                     │
│ • 论点完好；维持买入评级                                          │
├─────────────────────────────────────────────────────────────────┤
│ 业绩快照                                                         │
│ ┌─────────────┬──────────┬──────────┬──────────┐               │
│ │ 指标        │ 实际     │ 一致预期 │ 超/不及  │               │
│ │ 收入        │ $X.XXB   │ $X.XXB   │ +X.X%    │               │
│ │ EPS         │ $X.XX    │ $X.XX    │ +$X.XX   │               │
│ │ 毛利率      │ XX.X%    │ XX.X%    │ +XXbps   │               │
│ └─────────────┴──────────┴──────────┴──────────┘               │
└─────────────────────────────────────────────────────────────────┘

第2-3页：详细业绩
- 逐分部分析
- 地域细分
- 超预期/不及预期关键驱动因素

第4-5页：指标与指引
- 利润率分析
- 全年指引比较
- 更新季度预估

第6-7页：论点更新
- 变化内容
- 风险与催化剂
- 投资推荐

第8-10页：估值
- 更新DCF/可比公司（如有重大变化）
- 目标价论证
- 情景分析

来源部分（含可点击超链接）：
- 财报发布：[超链接]
- 季报：[公告查询链接]
- 财报电话会议记录：[超链接]
- 一致预期：Wind截至[日期]
```

## 质量清单

交付前：
- [ ] 财报数据来自最新季度（非过时）
- [ ] 超预期/不及预期用具体数字量化
- [ ] 所有图表已嵌入（共8-12张）
- [ ] 来源部分含可点击超链接
- [ ] 每个图表/表格有来源引用
- [ ] 指引变化清晰记录
- [ ] 评级和目标价在开头说明
- [ ] 8-12页，3,000-5,000字
