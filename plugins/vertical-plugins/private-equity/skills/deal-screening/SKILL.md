# 交易筛选

## 工作流程

### 第一步：提取交易事实

从提供的CIM、摘要或描述中提取：

- **公司**：名称、所在地、行业/细分行业
- **描述**：做什么（1-2句话）
- **财务数据**：收入、EBITDA、利润率、增长率
- **交易类型**：平台、补充收购、资本重组、少数股权、分拆
- **要价/估值**：倍数、企业价值（如注明）
- **卖方动机**：为什么现在出售
- **管理层**：留任还是退出
- **关键客户**：集中度风险
- **主要风险**：明显的风险标记

### 第二步：对照标准筛选

对照基金的投资标准进行筛选（如不清楚则询问用户）：

| 标准 | 目标 | 实际 | 通过/不通过 |
|-----------|--------|--------|-----------|
| 收入范围 | | | |
| EBITDA范围 | | | |
| EBITDA利润率 | | | |
| 增长特征 | | | |
| 行业匹配 | | | |
| 地域 | | | |
| 交易规模/企业价值 | | | |
| 估值（x EBITDA） | | | |
| 客户集中度 | | | |
| 管理层延续性 | | | |

### 第三步：快速评估

提供三部分评估：

1. **结论**：通过 / 进一步尽调 / 直接放弃
2. **看多理由**（2-3条）：为什么这可能是一笔好交易
3. **看空理由**（2-3条）：主要风险和担忧
4. **关键问题**：初次沟通中需要回答的问题

### 第四步：输出

一页纸筛选备忘录，适合与合伙人分享或投委会快速审查。

## 重要说明

- 速度很重要——筛选应该几分钟完成，而不是几小时
- 对风险标记要直截了当。不要掩盖担忧
- 如果财务数据看起来不一致或不完整，明确标记
- 如果是首次筛选，提前询问基金的标准
- 确认后将筛选标准保存到记忆中，供未来交易使用
