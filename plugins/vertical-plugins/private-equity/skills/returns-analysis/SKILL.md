# 收益分析

## 工作流程

### 第一步：收集交易输入

向用户询问（或从前期分析中提取）：

**入场：**
- 入场EBITDA（LTM或NTM）
- 入场倍数（EV/EBITDA）
- 企业价值
- 交割时净债务
- 股权投入规模
- 交易费用和开支

**融资：**
- 优先债务（x EBITDA、利率、还款计划）
- 次级债务/夹层融资（如有）
- 入场时总杠杆（x EBITDA）
- 股权出资

**运营假设：**
- 收入增长率（年度）
- EBITDA利润率轨迹
- 资本支出占收入比例
- 营运资金变化
- 偿债计划

**退出：**
- 持有期（年）
- 退出倍数（EV/EBITDA）
- 退出EBITDA（根据增长假设计算）

## 关键公式

- **MOIC** = 退出股权价值 / 股权投入
- **IRR** = 求解r：股权投入 × (1 + r)^n = 退出股权价值（调整中间现金流）
- **收益归因**：
  - 增长：(退出EBITDA - 入场EBITDA) × 退出倍数 / 股权
  - 倍数：(退出倍数 - 入场倍数) × 入场EBITDA / 股权
  - 杠杆：持有期内偿债额 / 股权

## 重要说明

- 始终同时展示费用/收益分成前后的收益（如适用）
- 管理层跟投和联合投资会改变股权投入——如有相关情况需询问
- 股息再资本化或中期分配对IRR影响显著——如有计划需纳入
- 不要忘记交易成本（通常为企业价值的2-4%）——它们会降低第1天的股权价值
- 税务考虑（资产交易 vs. 股权交易、338(h)(10)选择）会实质性影响税后收益
