# 投资委员会备忘录

## 工作流程

### 第一步：收集输入

从用户（或会话中的前期分析）收集：

- 公司概况和业务描述
- 行业/市场背景
- 历史财务数据（3-5年）
- 管理层评估
- 交易条款（价格、结构、融资）
- 尽调发现（商业、财务、法律、运营）
- 价值创造计划/百日计划
- 收益分析（基准、上行、下行）

### 第二步：起草备忘录结构

标准IC备忘录格式：

**一、执行摘要**（1页）
- 公司描述、交易逻辑、关键条款
- 推荐意见和核心收益数据
- 前3大风险及缓解措施

**二、公司概况**（1-2页）
- 业务描述、产品/服务
- 客户群和市场推广方式
- 竞争定位
- 管理团队

**三、行业与市场**（1页）
- 市场规模和增长
- 竞争格局
- 长期趋势/有利因素
- 监管环境

**四、财务分析**（2-3页）
- 历史表现（收入、EBITDA、利润率、现金流）
- 盈利质量调整
- 营运资金分析
- 资本支出需求

**五、投资主题**（1页）
- 为什么这是一笔有吸引力的投资（3-5个支柱）
- 价值创造杠杆（有机增长、利润率提升、并购、倍数扩张）
- 百日优先事项

**六、交易条款与结构**（1页）
- 企业价值和隐含倍数
- 资金来源与用途
- 资本结构/杠杆
- 关键法律条款

**七、收益分析**（1页）
- 基准、上行和下行情景
- 各情景下的IRR和MOIC
- 驱动收益的关键假设
- 敏感性分析

**八、风险因素**（1页）
- 按严重程度和可能性排序的关键风险
- 每项风险的缓解措施
- 交易终止风险（如有）

**九、推荐意见**
- 明确推荐：推进 / 放弃 / 附条件推进
- 关键条件或下一步

### 第三步：输出格式

- 默认：专业排版的Word文档（.docx）
- 备选：Markdown格式便于快速审阅
- 包含财务和收益的表格，而不仅仅是文字

## 重要说明

- IC备忘录应客观平衡——如实呈现看多和看空案例
- 不要弱化风险。IC成员总会发现风险；可信度很重要
- 如果用户提供公司的标准备忘录模板，使用该模板
- 财务表格应保持一致——检查EBITDA桥、资金来源与用途平衡、收益计算的一致性
- 缺少输入时应询问，而不是对交易条款或收益做假设
