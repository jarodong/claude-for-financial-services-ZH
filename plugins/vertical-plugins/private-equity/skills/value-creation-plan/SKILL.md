# 价值创造计划

## 工作流程

### 第一步：基线评估

了解起始点：
- 当前收入、EBITDA和利润率
- 组织架构和能力
- 各职能的关键运营指标
- 管理团队的优势和短板
- 尽调中已识别的速赢机会

### 第二步：价值创造杠杆

将所有杠杆映射到持有期内的EBITDA桥：

#### 收入增长杠杆
- **有机增长**：提价、销量增长、市场扩展
- **交叉销售/追加销售**：向现有客户销售新产品
- **新市场进入**：地域扩展、新垂直领域、新渠道
- **销售团队效能**：招聘销售代表、提高转化率、缩短销售周期
- **并购/补充收购**：补充性收购以增加收入和能力

对每个杠杆：
- 当前状态 → 目标状态
- 收入影响（人民币）
- 影响时间表
- 所需投资
- 置信度（高/中/低）

#### 利润率提升杠杆
- **定价优化**：提价、组合调整、捆绑销售
- **销货成本降低**：采购节省、供应商整合、自动化
- **运营费用优化**：管理费用降低、共享服务、离岸外包
- **技术投资**：自动化、系统集成、数据分析
- **规模杠杆**：随收入增长的固定成本杠杆

#### 战略/倍数扩张
- **平台构建**：补充收购、整合
- **经常性收入转型**：从项目制转向经常性/订阅制
- **市场定位**：品类领导力、品牌建设
- **管理层升级**：关键招聘以专业化运营
- **ESG/治理**：董事会组建、报告改进

### 第四步：百日计划

优先安排交割后前100天：

**第1-30天：稳定与评估**
- 管理层对齐和留任（签署雇佣协议、设定薪酬）
- 速赢——定价、明显降本、低垂果实
- 各职能的详细运营评估
- 客户沟通计划
- 建立报告和KPI仪表板

**第31-60天：规划与启动**
- 确定战略计划并向组织传达
- 启动前3-5个价值创造项目
- 开始补充并购管线开发
- 招聘关键缺口岗位
- 实施新的报告节奏（每周快报、每月回顾、每季度董事会）

**第61-100天：执行与衡量**
- 速赢项目的首批成果
- 首次带运营指标的董事会会议
- 各价值创造杠杆的进展报告
- 根据早期经验调整计划

### 第五步：KPI仪表板

定义跟踪价值创造的指标：

### 第六步：输出

- Word文档或PowerPoint，包含：
  - 执行摘要（1页）
  - EBITDA桥图表
  - 价值创造杠杆详情（每个杠杆1页）
  - 百日计划时间表
  - KPI仪表板
  - 责任矩阵（谁负责什么）
- 支持EBITDA桥的Excel模型

## 重要说明

- 对时间表要现实——大多数PE价值创造需要12-24个月才能在财务数据中体现
- 速赢对建立势头和可信度很重要，但不要过度聚焦降本而牺牲增长
- 管理层认同至关重要——共同制定计划，而非强加
- 跟踪项目级损益影响，而不仅仅是顶层EBITDA——你需要知道什么在起作用
- 补充并购通常是最主要的价值创造杠杆——第1天就开始建立管线
- 始终用运营合伙人或行业专家对假设进行压力测试
