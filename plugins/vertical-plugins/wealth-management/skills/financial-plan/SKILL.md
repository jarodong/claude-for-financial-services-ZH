# 财务规划

## 工作流程

### 第一步：客户档案

收集或确认：
- **人口统计**：年龄、配偶年龄、受抚养人、预期寿命假设
- **就业**：当前收入、预期加薪、目标退休年龄
- **账户**：所有投资账户及其余额和资产配置
- **收入来源**：工资、奖金、租金收入、社保预估、养老金
- **支出**：当前年度支出、预期变化（房贷还清、子女独立）
- **负债**：房贷、学生贷款、其他债务
- **保险**：人寿、残疾、长期护理、医疗
- **遗产**：遗嘱、信托、受益人指定、赠与策略

### 第二步：现金流分析

构建年度现金流预测：

| 年份 | 年龄 | 总收入 | 税款 | 生活支出 | 储蓄 | 净现金流 |
|------|-----|-------------|-------|-----------------|---------|--------------|
| | | | | | | |

关键输入：
- 通胀率假设（通常2.5-3%）
- 税率（边际税率和有效税率）
- 储蓄率和储蓄去向（税前、个人养老金账户（税后缴费型）、应税账户）

### 第三步：退休预测

**积累期：**
- 当前投资组合价值
- 年度缴款（企业年金、个人养老金账户、应税账户）
- 各资产类别的预期收益
- 蒙特卡洛模拟：不同支出水平下的成功概率

**分配期：**
- 退休所需年度支出（当前人民币→通胀调整后）
- 社保开始年龄和福利
- 养老金收入（如有）
- 投资组合提取顺序和比例
- 最低领取要求

**关键输出：**
- 退休时预估投资组合价值
- 可持续提取率
- 资金不耗尽的概率（目标>85%）
- "如果"情景：提前退休、市场下跌、更高支出

### 第四步：目标专项分析

#### 教育基金
- 子女年龄和目标大学入学时间
- 当前529计划余额
- 目标资助水平（公立 vs. 私立、4年制 vs. 研究生）
- 达到目标所需月储蓄额
- 助学金考虑

#### 遗产规划
- 当前遗产价值和预估增长
- 遗产税风险敞口（联邦和州）
- 已设立的信托架构
- 赠与策略（年度免税额、终身免税额使用）
- 慈善捐赠计划
- 受益人审查

#### 风险管理
- 人寿保险需求分析（收入替代、债务偿还、教育基金）
- 残疾保险充分性
- 长期护理规划
- 伞形责任险覆盖

### 第五步：情景建模

运行关键情景：

| 情景 | 成功概率 | 90岁时投资组合 | 备注 |
|----------|----------------------|-----------------|-------|
| 基准情景 | | | |
| 提前2年退休 | | | |
| 第1年市场下跌20% | | | |
| 更高支出（+20%） | | | |
| 配偶活到95岁 | | | |
| 长期护理事件 | | | |

### 第六步：建议

按优先级排序的行动事项：
1. 储蓄率调整
2. 资产配置调整
3. 税务优化（个人养老金账户（税后缴费型）转换、税收亏损收割、资产配置位置）
4. 填补保险缺口
5. 遗产文件更新
6. 受益人指定审查

### 第七步：输出

- 财务规划文档（Word/PDF，15-25页）
- 现金流预测电子表格（Excel）
- 退休预测图表
- 目标资助分析
- 情景对比表
- 行动事项清单

## 重要说明

- 财务规划是动态文档——每年或重大生活事件后回顾和更新
- 收益假设要保守——高估收益会带来虚假信心
- 税务规划与投资收益同等重要——对每项建议建模税务影响
- 社保领取时间是重要杠杆——建模62岁、67岁和70岁开始领取的情景
- 始终对规划进行压力测试——只在基准情景下可行的规划不是好规划
- 合规：确保建议符合适当性/信托标准
