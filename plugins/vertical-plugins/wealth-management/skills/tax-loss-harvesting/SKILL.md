# 税收亏损收割

## 工作流程

### 第一步：识别候选标的

扫描应税账户中持有未实现亏损的持仓：

| 证券 | 资产类别 | 成本基础 | 当前市值 | 未实现亏损 | 持有期 | 亏损% |
|----------|-----------|-----------|---------------|-----------------|---------------|--------|
| | | | | | 短期/长期 | |

**优先排序：**
1. 最大绝对亏损（最大税收收益）
2. 优先处理短期亏损（抵消按普通所得税率征税的短期收益）
3. 亏损百分比最大的持仓（恢复可能性较低）

### 第二步：收益/亏损预算

计算客户的税务状况：

| 类别 | 金额 |
|----------|--------|
| 年初至今已实现短期收益 | |
| 年初至今已实现长期收益 | |
| 年初至今已实现亏损 | |
| 净收益/（亏损）头寸 | |
| 往年结转亏损 | |
| **目标收割金额** | |

**税收节省预估：**
- 短期亏损 × 边际普通所得税率
- 长期亏损 × 资本利得税率
- 最多¥3,000净亏损可抵扣普通收入
- 超额部分结转

### 第三步：替代证券

对每个收割候选，建议满足以下条件的替代品：
- 维持相似的市场敞口（相同资产类别、行业、地域）
- 不"实质相同"（洗售规则）
- 具有相似的风险/收益特征

| 卖出 | 替代为 | 原因 | 跟踪误差风险 |
|------|-------------|--------|-------------------|
| 沪深300ETF (510300) | 沪深300ETF | 同一指数，不同基金公司 | 极小 |
| 海外股票ETF | 全球股票ETF | 相似敞口，不同指数 | 低 |
| 个股ABC | 科技行业ETF | 更广泛敞口，无洗售风险 | 中等 |

### 第四步：洗售检查

执行前，确认无洗售：

- 检查家庭中所有账户（应税账户、个人养老金账户、个人养老金账户（税后缴费型）、配偶账户）
- 30天回溯：过去30天内是否购买了实质相同的证券？
- 30天前瞻：30天内禁止回购同一证券
- 检查股息再投资计划（DRIP）是否可能触发洗售
- 记录每笔交易的洗售窗口

| 卖出证券 | 洗售窗口开始 | 窗口结束 | DRIP是否激活 | 风险 |
|--------------|----------------------|-----------|-------------|------|
| | | | | |

### 第五步：执行计划

| 交易编号 | 账户 | 操作 | 证券 | 份额 | 预估收入 | 预估亏损 | 替代品 | 备注 |
|---------|---------|--------|----------|--------|--------------|-----------|-------------|-------|
| | | 卖出 | | | | | | |
| | | 买入 | | | | | | |

**摘要：**
- 预估收割总亏损：¥
- 预估税收节省：¥（边际税率%）
- 对投资组合的净影响：极小（替代证券维持敞口）
- 洗售窗口管理：[日期]

### 第六步：收割后跟踪

30天后，可选：
- 换回原始证券（如更偏好）
- 维持替代证券（如无换回理由）
- 更新成本基础记录
- 记录用于税务申报

### 第七步：输出

- 收割机会清单（Excel）
- 交易执行表
- 洗售跟踪日历
- 税收节省预估摘要
- 替代证券选择理由

## 重要说明

- 洗售规则很严格——违规会导致亏损不允许抵扣且调整成本基础
- 实质相同指同一证券，而非同一资产类别——跟踪不同指数的ETF通常没问题
- 始终跨所有家庭账户协调，包括退休账户
- 考虑长期成本基础降低——收割会重置成本基础，意味着未来更多收益
- 年末是收割旺季，但全年都存在机会
- 12月共同基金资本利得分配可能产生额外的收割紧迫性
- 记录一切用于税务申报和合规
- 并非所有亏损都值得收割——交易成本和跟踪误差有实际成本
