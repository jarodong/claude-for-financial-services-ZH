# 投资组合再平衡

## 工作流程

### 第一步：当前状态

对每个账户，获取：
- 账户类型（应税账户、个人养老金账户、个人养老金账户（税后缴费型）、企业年金）
- 当前市值的持仓
- 成本基础（应税账户）
- 每个持仓的未实现收益/损失

### 第二步：偏离分析

将当前配置与IPS目标对比：

| 资产类别 | 目标% | 当前% | 偏离 | 超出/不足金额 |
|------------|----------|-----------|-------|-------------|
| 美国大盘股票 | | | | |
| 美国中小盘 | | | | |
| 国际发达市场 | | | | |
| 新兴市场 | | | | |
| 投资级债券 | | | | |
| 高收益/信用 | | | | |
| TIPS/通胀保护 | | | | |
| 另类投资 | | | | |
| 现金 | | | | |

标记超过再平衡带（通常+/-3-5%）的持仓。

### 第三步：交易建议

生成将配置恢复到目标的交易：

**税务感知再平衡规则：**
- 优先在税收优惠账户（个人养老金账户、个人养老金账户（税后缴费型））中再平衡——无税务后果
- 在应税账户中，避免卖出有大额短期收益的持仓
- 在再平衡的同时尽可能收割亏损
- 注意洗售规则（30天窗口）——跨所有账户
- 考虑将新增缴款导向低配资产类别，而非交易

**交易清单：**

| 账户 | 操作 | 证券 | 份额/金额 | 原因 | 税务影响 |
|---------|--------|----------|----------|--------|-----------|
| | 买入/卖出 | | | 再平衡/TLH | 短期收益/长期收益/亏损 |

### 第四步：资产配置位置审查

优化哪些资产持有在哪类账户中：
- **税收递延账户（个人养老金账户/企业年金）**：债券、REITs、高换手率基金（税收拖累最大）
- **个人养老金账户（税后缴费型）**：预期收益最高的资产（免税增长）
- **应税账户**：税收效率高的股票（指数基金、ETF、市政债）、税收亏损收割候选

### 第五步：实施

- 按账户汇总交易
- 预估交易成本
- 预估税务影响（已实现收益/损失）
- 对配置偏离的净影响

### 第六步：输出

- 偏离分析表
- 推荐交易清单（Excel）
- 税务影响摘要
- 前后配置对比

## 重要说明

- 不要为再平衡而再平衡——带内的小幅偏离没问题
- 应税账户中税务成本可能超过再平衡收益——计算盈亏平衡点
- 交易前考虑待处理现金流（缴款、提取、最低领取要求）
- 检查客户特定限制（ESG、集中持仓、锁定期）
- 为每笔交易记录理由以备合规
- 洗售规则跨账户适用——协调整个家庭的交易
