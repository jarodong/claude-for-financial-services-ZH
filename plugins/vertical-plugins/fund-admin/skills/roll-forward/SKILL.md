# 滚动调节

给定一个科目（或科目组）、实体和期间，生成从期初到期末的滚动调节表。

## 结构

```
期初余额（上期结账数）                      X
  + 新增 / 新发生活动                       A
  + 本期已提计提                            B
  − 上期计提转回                           (C)
  − 付款 / 结算                            (D)
  ± 重分类 / 调整                           E
  ± 汇率折算                                F
期末余额（期末 GL 余额）                    Y
```

## 逐行对应

- **期初**——上期结账包，或上期末 GL 余额。
- **每行发生额**——通过 internal-gl MCP 进行 GL 查询（科目 + 日期范围 + 凭证来源筛选）。注明查询引用。
- **期末**——期末 GL 余额。

调节表**必须平衡**：`X + A + B − C − D + E + F = Y`。如不平衡，差额为未解释项目——需揭示而非人为调节。

## 输出

滚动调节表，附"对应来源"列注明各行对应的 GL 查询或文档，加平衡检查（通过/未通过及未解释差额）。
