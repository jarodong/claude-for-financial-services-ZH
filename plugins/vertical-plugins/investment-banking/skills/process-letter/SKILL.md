# 流程函

## 工作流程

### 第一步：确定函件类型

- **初始流程函**：随 Teaser/CIM 一同发送，概述流程和 IOI 要求
- **IOI 指引**：首轮意向书的具体要求
- **第二轮/最终报价函**：尽调后提交约束性报价的指引
- **管理层会议邀请**：现场管理层演示的安排

### 第二步：初始流程函 / IOI 指引

**标题：**
- 日期、交易代号
- "保密"
- 致潜在买家

**章节：**

1. **引言**：机会和卖方目标的简要概述
2. **流程概述**：时间表、关键日期、预期轮次数
3. **IOI 要求**：初步意向书应包含：
   - 拟议估值区间（企业价值）
   - 对价形式（现金、股票、盈利支付、滚转投资）
   - 融资来源和确定性
   - 关键尽调需求
   - 指示性交割时间表
   - 任何条件或前提条件
   - 买家简介和战略逻辑
4. **提交详情**：发送地址、截止日期（日期和时间）、格式
5. **保密提醒**：引用 NDA、数据室访问
6. **联系方式**：负责咨询的银行家联系方式

### 第三步：最终报价 / 第二轮函

在 IOI 基础上的额外要求：

1. **收购协议标稿**：提供 SPA/APA 草稿并要求标注意见
2. 详细融资承诺：需要已承诺的融资函
3. **剩余尽调事项**：明确预期的确认性尽调内容
4. **排他条款**：排他期的时长和条件
5. **监管分析**：反垄断申报要求和时间表
6. **核心人员条款**：雇佣协议、薪酬、滚转股权
7. **约束性 vs. 非约束性**：明确本阶段哪些内容具有约束力
8. **评估标准**：报价的评估方式（价格、确定性、速度、匹配度）

### 第四步：管理层会议邀请

1. **安排**：日期、时间、地点（或视频链接）、时长
2. **出席人员**：公司方出席演讲人员、买方应出席人员
3. **议程**：典型管理层演示议程（概述、财务、运营、增长、问答）
4. **基本规则**：不得录音、保密要求、提问格式
5. **材料**：将分发的内容（演示文稿、数据室访问）
6. **后续**：会后提交补充问题的流程

### 第五步：输出

- Word 文档（.docx），专业函件格式
- 律所信头占位符
- 供客户审查的修订版

## 重要说明

- 流程函为整个交易定调 —— 要清晰、专业、有条理
- 截止日期应坚定但合理 —— IOI 通常2-3周，最终报价3-4周
- 始终包含评估标准 —— 买家想知道他们将如何被评判
- 与法务沟通函件中的任何陈述或承诺
- 发送前客户应审核和批准 —— 他们可能想调整语气或条款
- 记录每封函件的接收人和时间 —— 这将成为流程跟踪器
