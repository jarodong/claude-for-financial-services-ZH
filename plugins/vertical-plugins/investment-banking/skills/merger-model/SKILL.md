# 并购模型

## 工作流程

### 第一步：收集输入数据

**收购方：**
- 公司名称、当前股价、流通股数
- LTM 和 NTM EPS（企业会计准则口径和扣非后口径）
- P/E 倍数
- 税前债务成本、税率
- 账面现金、现有债务

**标的方：**
- 公司名称、当前股价、流通股数（如为上市公司）
- LTM 和 NTM EPS 或净利润
- 企业价值或股权价值

**交易条款：**
- 每股报价（或相对当前价格的溢价）
- 对价组合：% 现金 vs. % 股票
- 为现金部分筹集的新债务
- 预期协同效应（收入和成本）及分阶段实现时间表
- 交易费用和融资成本
- 预期交割日期

### 第七步：输出

- Excel 工作簿，包含：
  - 假设标签页
  - 资金来源与用途
  - 备考利润表
  - 增厚/稀释摘要
  - 敏感性分析表
  - 盈亏平衡分析
- 一页并购后果摘要，用于推介材料

## 重要说明

- 始终同时展示企业会计准则口径和扣非后口径（现金）EPS（如相关）
- 股票交易：使用收购方当前价格计算换股比例，注意新股稀释
- 包含购买对价分配 —— 商誉和无形资产摊销对基本每股收益有重要影响
- 协同效应分阶段实现至关重要 —— 第1年通常仅为满负荷协同效应的25-50%
- 不要忘记放弃的现金利息收入和新增债务的利息支出
- 协同效应和利息调整的税率应与收购方边际税率一致
