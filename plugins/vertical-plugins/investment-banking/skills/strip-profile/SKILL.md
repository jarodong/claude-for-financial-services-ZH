## 工作流程

### 1. 明确需求
- **询问用户**：单页还是多页（3-4页）？
- **询问用户**：有无特定重点领域或主题需要强调？
- **仅在用户确认后**，才进行研究

### 2. 研究与规划
**数据来源：**
- **主要来源**：公司文件（巨潮资讯网 - "公司业务概况"、MD&A）、投资者演示、公司网站
- **市场数据**：Wind、iFinD、Choice（价格、股数、市值、净债务、EV、股东结构）
- **预测数据**：Wind / iFinD 的 NTM 收入、EBITDA、EPS 一致预期
- **新闻**：最近90天的新闻稿、并购活动、指引变更

**必需指标：**
- **财务数据**：±3年的收入、EBITDA、利润率（%）、EPS、FCF
- **估值**：市值、EV、EV/收入、EV/EBITDA、P/E 倍数
- **增长率**：同比增长率（%）
- **股东结构**：前5大股东及持股比例
- **业务板块**：产品组合和/或地理分布（%分解）

**标准化：**
- 将所有金额转换为统一货币
- 规模统一（全程使用 百万元 或 亿元，不混用）

**构建前：**
- 在对话中打印大纲，每项4-5个要点（实际数字，非占位符）
- 打印样式选择：字体、颜色（十六进制代码）、每组数据的图表类型
- 获得用户确认："这个大纲和视觉策略符合你的预期吗？"

### 3. 逐页创建
**关键提示：你必须一次创建一页，获得用户批准后再创建下一页。**

**对每一页：**
1. 仅使用 PptxGenJS 创建这一页
2. **必须：转换为图片进行审查** - 你必须将幻灯片转换为图片以便视觉验证：
   ```bash
   soffice --headless --convert-to pdf presentation.pptx
   pdftoppm -jpeg -r 150 -f 1 -l 1 presentation.pdf slide
   ```
3. **必须进行视觉审查**：在继续之前你必须仔细检查渲染后的幻灯片图片：
   - **文字重叠检查**：扫描每个文字元素 —— 标签、要点或标题是否相互碰撞？
   - **文字截断检查**：是否有文字在边界处被截断？所有单词是否完整可见？
   - **图表边界检查**：图表是否保持在容器内？所有坐标轴标签是否完整可见？
   - **象限完整性**：一个象限的内容是否渗入相邻象限？
4. **如检测到任何重叠或截断**：按以下策略顺序立即修复：
   - **首先**：减小字号（下调1-2pt）
   - **其次**：缩短文字（缩写、删除较不重要的信息）
   - **第三**：调整元素位置或容器大小
   - **重新渲染并再次验证** - 在所有文字完整显示前不要继续
5. 向用户展示幻灯片图片和下载链接
6. **停止并等待用户明确批准**后再创建下一页。用户确认前不要继续。

**你必须在每一页检查以下具体问题：**
- 表格行与下方文字碰撞
- 图表 x 轴标签在底部被截断
- 长要点折行进入相邻内容
- 象限内容渗入相邻象限
- 标题文字与下方内容重叠
- 图例文字与图表元素重叠
- 脚注/来源文字与主内容碰撞

## 幻灯片格式要求

### 信息密度至关重要

**首要目标是最大化信息密度。** 高管应在30秒内理解整个公司故事。将每个象限填满至容量上限。

**每个象限的目标：**
- **公司概述**：最少6-8个要点（总部、成立年份、员工数、CEO/CFO、市值、股票代码、行业、关键数据）
- **业务与定位**：6-8个要点（收入驱动因素、产品、市场份额%、竞争护城河、客户数量、地理分布）
- **关键财务数据**：含8-10行的表格 或 图表 + 4-5个关键指标（收入、EBITDA、利润率、EPS、FCF、增长率、估值倍数）
- **第四象限**：5-7个要点（股东结构%、近期并购、发展动态、催化剂）

**信息密集技巧：**
- 合并相关事实："总部：德克萨斯州奥斯汀；成立：2003年；14万员工"
- 始终包含数字："50亿元 收入" 而非 "大量收入"
- 添加背景："EBITDA 利润率：25%（行业平均18%）"
- 包含同比变化："收入：1.25亿元（同比+28%）"
- 使用百分比："企业业务：占收入62%"

**如象限显得空旷，添加更多：**
- 按%分解的板块结构
- 地理收入分布
- 客户集中度（前10大 = X%）
- 近期合同签订及金额
- 指引 vs. 一致预期
- 内部持股%

**行间距 - 每个部分使用单个文本框：**
```python
def add_section(slide, x, y, w, header_text, bullets, header_size=10, bullet_size=8):
    """标题 + 要点在单个文本框中，自然间距"""
    tb = slide.shapes.add_textbox(x, y, w, Inches(len(bullets) * 0.18 + 0.3))
    tf = tb.text_frame
    tf.word_wrap = True

    # 标题段落
    p = tf.paragraphs[0]
    p.text = header_text
    p.font.bold = True
    p.font.size = Pt(header_size)
    p.font.color.rgb = RGBColor(0, 51, 102)
    p.space_after = Pt(6)  # 标题后小间距

    # 要点段落
    for bullet in bullets:
        p = tf.add_paragraph()
        p.text = bullet
        p.font.size = Pt(bullet_size)
        p.space_after = Pt(3)
    return tb
```

**关键间距原则：**
- 将标题 + 要点放在同一文本框（不要单独的标题文本框）
- 标题后使用 `space_after = Pt(6)`，要点间使用 `Pt(3)`
- 不要硬编码间距 —— 让段落间距自然处理
- 如内容溢出，将字号减小1pt 而非删除内容

---

- **3-4页密集幻灯片** - 使用象限、分栏、表格、图表
- **正文全部使用要点** - 绝不使用段落。**每个部分使用一个文本框，所有要点放在其中** - 不要为每个要点创建单独的文本框。使用 PptxGenJS 要点格式：
  ```javascript
  // 正确：单个文本框含要点列表 - 每个数组项成为一个要点
  // 位于左上象限（公司概述）- 标题和强调条之后
  slide.addText(
    [
      { text: '总部：广东深圳；成立：1995年', options: { bullet: { indent: 10 }, breakLine: true } },
      { text: '员工：约90万人，业务覆盖全球70+国家', options: { bullet: { indent: 10 }, breakLine: true } },
      { text: '董事长：王传福；董秘：李黔', options: { bullet: { indent: 10 }, breakLine: true } },
      { text: '市值：8500亿元（全球第6）', options: { bullet: { indent: 10 }, breakLine: true } },
      { text: '业务板块：汽车（85%）、电池（10%）、其他（5%）', options: { bullet: { indent: 10 } } }
    ],
    { x: 0.45, y: 0.95, w: 4.5, h: 2.6, fontSize: 11, fontFace: 'Arial', valign: 'top', paraSpaceAfter: 6 }
  );

  // 错误：多个单独的文本框各放一个要点 - 导致对齐问题
  // slide.addText('总部：广东深圳', { x: 0.5, y: 1.0, bullet: true });
  ```

  **要点格式提示：**
  - `bullet: { indent: 10 }` - 控制要点缩进（越小越紧凑）
  - `paraSpaceAfter: 6` - 每段后的间距（磅）
  - 将多个相关事实打包到每个要点中（如"总部：广东深圳；成立：1995年"）
  - 包含具体数字和百分比以提高信息密度
- 标题使用**标题大写**（非全大写），左对齐
- **全文统一字体**，包括表格
- **公司品牌色** - 创建幻灯片前你必须通过网络搜索研究实际品牌色。不要猜测或假设颜色。
- **遵循提供的品牌指南**

## 首页布局

必须通过高管的"30秒理解测试"。

### 幻灯片设置（关键）
**使用4:3宽高比**（标准投行推介材料格式）：
```javascript
const pptx = new pptxgen();
pptx.layout = 'LAYOUT_4x3';  // 10" 宽 × 7.5" 高 - 必须使用此设置
```

### 幻灯片坐标系
PptxGenJS 使用英寸。4:3幻灯片 = **10" 宽 × 7.5" 高**。
- **x**：距左边缘的水平位置（0 = 左，10 = 右）
- **y**：距上边缘的垂直位置（0 = 上，7.5 = 下）
- **内容必须保持在边界内** - 四边留0.3"边距

### 首页定位（英寸）
```
┌─────────────────────────────────────────────────────────────────┐
│ y=0.2  标题：公司名称（股票代码）                             │
├────────────────────────────┬────────────────────────────────────┤
│ y=0.6  公司概述            │ y=0.6  业务与定位                  │
│ x=0.3, w=4.7               │ x=5.0, w=4.7                       │
│ h=3.0                      │ h=3.0                              │
├────────────────────────────┼────────────────────────────────────┤
│ y=3.7  关键财务数据        │ y=3.7  股价/近期动态               │
│ x=0.3, w=4.7               │ x=5.0, w=4.7                       │
│ h=3.5                      │ h=3.5                              │
└────────────────────────────┴────────────────────────────────────┘
                                                            y=7.5
```

### 标题区域（y=0.2）
**公司名称（股票代码）** - 示例：`比亚迪 (002594.SZ)`
```javascript
slide.addText('比亚迪 (002594.SZ)', { x: 0.3, y: 0.2, w: 9.4, h: 0.35, fontSize: 18, bold: true });
```

### 四象限布局（y=0.6 到 y=7.2）

| 象限 | 位置 | 内容 |
|----------|----------|---------|
| **1** | x=0.3, y=0.6, w=4.7, h=3.0 | **公司概述**：总部、成立年份、关键数据、业务摘要（4-5个要点） |
| **2** | x=5.0, y=0.6, w=4.7, h=3.0 | **业务与定位**：收入驱动因素、产品/服务、竞争地位、增长驱动因素（4-5个要点） |
| **3** | x=0.3, y=3.7, w=4.7, h=3.5 | **关键财务数据**：收入、EBITDA、利润率、EPS、FCF + 估值（市值、EV、倍数）— **表格或图表，不同时使用** |
| **4** | x=5.0, y=3.7, w=4.7, h=3.5 | **上市公司**：1年股价图表 + 大股东。**非上市公司**：近期动态或股东结构/并购历史 |

### 字号 - 使用以下精确值
| 元素 | 大小 | 说明 |
|---------|------|-------|
| 幻灯片标题 | 24pt | 粗体，公司品牌色 |
| 象限标题 | 14pt | 粗体，带强调条 |
| 正文/要点 | 11pt | 常规粗细 |
| 表格文字 | 10pt | 密集表格用9pt |
| 图表标签 | 9pt | 保持标签简短 |
| 来源/脚注 | 8pt | 幻灯片底部 |

**关键提示：如文字溢出，将字号减小1pt并重新渲染。**

### 视觉强调（必须）
每个象限标题左侧必须有彩色强调条：
```javascript
// 为象限标题添加强调条
slide.addShape(pptx.shapes.RECTANGLE, {
  x: 0.3, y: 0.6, w: 0.08, h: 0.25,
  fill: { color: 'E31937' }  // 使用公司品牌色
});
slide.addText('Company Overview', {
  x: 0.45, y: 0.6, w: 4.5, h: 0.3, fontSize: 14, bold: true, fontFace: 'Arial'
});
```

**需包含的视觉元素：**
- 所有章节标题旁的强调条（品牌色）
- 上下象限之间的细分隔线
- 右上角的公司 logo（如有）
- 表格中使用细微网格线（浅灰 #CCCCCC）

### 首页格式
- **字体：Arial**（或按用户/品牌指南指定）
- **象限标题**：标题大写（非全大写），如"Company Overview"而非"COMPANY OVERVIEW"
- **要点**：开头粗体关键词，如"**Market Position:** Leading global manufacturer..."
- 仅白色背景 —— 无方框、填充或阴影
- 章节标题：粗体文字，按品牌指南设置样式
- 所有象限等大并对齐

## 图表（多页简介）

**多页简介**：包含2-3个实际的 PptxGenJS 图表。绝不使用占位符 div 或静态图片。

**单页简介**：财务数据使用表格（更节省空间）。仅在图表替代表格时才添加，不要同时使用。

| 数据类型 | 图表类型 |
|-----------|------------|
| 收入趋势 | 折线图或多列柱状图 |
| 地理分布 | 水平条形图 |
| 产品组合 | 带百分比的饼图 |
| 财务比较 | 柱状图 |
| 股价（1年日线） | 折线图 |

### 图表代码示例

**水平条形图（适合4:3幻灯片右下象限）：**
```javascript

```

**饼图（适合4:3幻灯片右下象限）：**

## 财务数据格式

**始终使用原生 PptxGenJS 表格或图表 - 绝不使用纯文本叙述或 HTML 表格。**

使用 `slide.addTable()` 展示财务数据（适合4:3幻灯片左下象限）：
```javascript
// 先添加带强调条的标题
slide.addShape(pptx.shapes.RECTANGLE, {
  x: 0.3, y: 3.7, w: 0.08, h: 0.25, fill: { color: 'E31937' }
});
slide.addText('Key Financials & Valuation', {
  x: 0.45, y: 3.7, w: 4.5, h: 0.3, fontSize: 14, bold: true, fontFace: 'Arial'
});

// 财务数据表
slide.addTable([
  [{ text: 'Metric', options: { bold: true, fill: '003366', color: 'FFFFFF' } },
   { text: 'FY24', options: { bold: true, fill: '003366', color: 'FFFFFF' } },
   { text: 'FY25E', options: { bold: true, fill: '003366', color: 'FFFFFF' } }],
  ['Revenue', '¥514亿', '¥542亿'],
  ['YoY Growth', '+6.0%', '+5.5%'],
  ['EBITDA', '¥89亿', '¥95亿'],
  ['EBITDA Margin', '17.3%', '17.5%'],
  ['EPS', '¥3.42', '¥3.75'],
  ['Market Cap', '¥1850亿', '—'],
  ['EV/EBITDA', '12.5x', '11.7x']
], {
  x: 0.45, y: 4.1, w: 4.3, h: 3.0,  // 位于左下象限标题下方
  fontFace: 'Arial', fontSize: 10,
  border: { pt: 0.5, color: 'CCCCCC' },
  valign: 'middle',
  colW: [1.8, 1.25, 1.25]  // 列宽
});
```

❌ **错误示例：** 纯文本如 `Note: FY2024 revenue growth +1.0%, Net Income 51亿元...`
❌ **错误示例：** 无法正确转换为 PowerPoint 的 HTML 表格

对于预测数据，在结构化表格中使用熊市/基准/牛市情景。
