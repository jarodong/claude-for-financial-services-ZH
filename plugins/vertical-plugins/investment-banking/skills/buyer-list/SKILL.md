# 买方名单

## 工作流程

### 第一步：了解标的公司

- 公司描述、行业及商业模式
- 收入、EBITDA 及增长特征
- 核心资产与能力（知识产权、客户关系、地理覆盖、团队）
- 预期估值区间
- 卖方偏好（战略买家 vs. 财务买家、管理层延续性、时间安排）

### 第二步：战略买家

在以下类别中识别战略收购方：

**直接竞争对手**
- 同一领域中可获得市场份额的公司
- 逻辑：收入协同效应、消除竞争、规模扩张

**邻近领域参与者**
- 可向标的公司领域拓展的邻近市场公司
- 逻辑：产品延伸、交叉销售、新市场进入

**纵向整合者**
- 可进行纵向整合的客户或供应商
- 逻辑：供应链控制、利润获取、战略锁定

**平台构建者**
- 通过并购在该领域构建平台的大型公司
- 逻辑：补强型收购、填补能力缺口

对每个战略买家进行评估：

| 买家 | 行业 | 收入 | 战略匹配度 | 财务能力 | 并购记录 | 可能性 | 优先级 |
|-------|--------|---------|--------------|-------------------|------------------|------------|----------|
| | | | 高/中/低 | | 活跃/一般/无 | | A/B/C |

### 第三步：财务投资人

识别 PE/财务买家：

**平台投资者**
- 在该行业寻找新平台的投资人
- 评估标准：基金规模、行业聚焦、交易规模区间

**补强收购买家**
- 已有投资组合公司可将标的作为补强收购的投资人
- 识别具体的投资组合公司及协同逻辑

**成长型股权**
- 适用于早期或高增长标的
- 少数股权 vs. 多数股权偏好

对每个投资人：

| 投资人 | 基金规模 | 行业聚焦 | 组合重叠 | 近期活动 | 优先级 |
|---------|-----------|-------------|-------------------|-----------------|----------|
| | | | | | A/B/C |

### 第四步：优先级分层

对买方名单进行分层：

- **第一梯队（5-10家）**：最高战略匹配度、有成功并购记录、逻辑清晰 —— 优先接触
- **第二梯队（10-15家）**：匹配度较好但不够明显 —— 第二轮接触
- **第三梯队（10-20家）**：有可能但概率较低 —— 流程需要扩大范围时接触

### 第五步：联系人对接

对每个第一梯队买家：
- 关键决策人（CEO、企业发展负责人、合伙人）
- 关系状态（已有关系、冷启动、需引荐）
- 已知偏好或限制（规模、地域、结构）
- 最佳接触渠道

### 第六步：输出

- Excel 工作簿，包含：
  - 战略买家表（按梯队排序）
  - 财务投资人表（按梯队排序）
  - 第一梯队联系人对接表
  - 汇总统计（按梯队、按类型汇总买方数量）
- 一页买方名单摘要，用于委托书或推介材料

## 重要说明

- 质量优于数量 —— 一份30-40家经过深入研究的买方名单，优于200家的大名单
- 研究近期并购动态 —— 刚在该领域完成交易的买家，要么仍想继续收购，要么已经子弹打光
- 检查直接竞争对手的反垄断风险 —— 标注可能面临监管问题的买家
- 财务投资人：检查基金成立年份和投资节奏 —— 接近投资期结束的基金可能更有动力
- 始终询问卖方是否有希望纳入或排除的买家
- 随流程推进更新名单 —— 根据反馈调整买方梯队
