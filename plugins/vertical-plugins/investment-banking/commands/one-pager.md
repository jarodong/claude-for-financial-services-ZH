# 单页公司简介命令

为推介材料和交易材料创建专业的单页公司简介。

## 工作流程

### 第一步：收集公司信息

如果提供了公司名称或股票代码，则直接使用。否则询问：
- "你想为哪家公司制作简介？"

### 第二步：检查可用的 PPT 模板 Skill

**首先，检查 skills 目录中是否存在 ppt-template 相关 skill：**

```bash
ls skills/ | grep -E "ppt-template|brand-guidelines"
```

如果存在模板 skill（如 `techcorp-ppt-template`、`gs-brand-guidelines`）：
1. 向用户列出可用模板
2. 询问使用哪个模板，或者使用简洁的专业格式
3. 使用 `skill: "[template-name]"` 加载所选模板 skill

如果不存在模板 skill，则询问：
- "你是否有品牌 PowerPoint 模板文件可供使用？如有，请提供路径。否则我将使用简洁的专业格式。"

如果提供了模板文件：
1. 分析模板结构以了解布局
2. 使用适合单页内容的布局

### 第三步：加载公司简介 Skill

使用 `skill: "strip-profile"` 执行简介创建：

1. **明确需求**：
   - 确认单页格式（one-pager）
   - 询问是否有特定重点领域

2. **研究公司数据**：
   - 公司概况（总部、成立年份、员工数、管理层）
   - 业务描述和定位
   - 关键财务数据（收入、EBITDA、利润率、增长率）
   - 估值指标（市值、EV、倍数）
   - 近期动态和新闻
   - 主要股东（上市公司）

3. **创建公司简介**：
   - 使用 4:3 宽高比（10" x 7.5"）
   - 四象限布局：
     - 左上：公司概述（要点）
     - 右上：业务与定位（要点）
     - 左下：关键财务数据（表格）
     - 右下：股价图表 + 股东 或 近期新闻
   - 应用公司品牌色
   - 章节标题添加强调条

### 第四步：视觉审查

创建幻灯片后：
1. 转换为图片进行审查
2. 检查文字重叠/截断问题
3. 验证所有数据已填充（无占位符）
4. 向用户展示预览以获取批准

### 第五步：交付输出

提供：
1. **PowerPoint 文件**（.pptx）—— 单页简介
2. **图片预览** —— 用于快速审查
3. **摘要** —— 包含的关键数据点

```
┌─────────────────────────────────────────────────────────────────┐
│ 公司名称（股票代码）                                       [Logo] │
├────────────────────────────┬────────────────────────────────────┤
│ 公司概述                   │ 业务与定位                         │
│ • 总部、成立年份、员工数   │ • 核心业务描述                     │
│ • CEO、CFO                 │ • 主要产品/服务                    │
│ • 市值、行业               │ • 竞争定位                         │
│ • 关键指标                 │ • 增长驱动因素                     │
├────────────────────────────┼────────────────────────────────────┤
│ 关键财务数据               │ 股价表现 / 股权结构                │
│ ┌──────────────────────┐   │ [1年股价图表]                     │
│ │ 指标   │ FY24 │ FY25E│   │                                    │
│ │ 收入   │ $XXB │ $XXB │   │ 主要股东：                        │
│ │ EBITDA │ $XXB │ $XXB │   │ • Vanguard: X.X%                  │
│ │ 利润率 │ XX%  │ XX%  │   │ • BlackRock: X.X%                 │
│ │ EV/EBITDA│ XXx │ XXx │   │ • State Street: X.X%              │
│ └──────────────────────┘   │                                    │
└────────────────────────────┴────────────────────────────────────┘
来源：公司公告、Wind / iFinD
```

## 质量清单

交付前检查：
- [ ] 所有 4 个象限填充了真实数据
- [ ] 无残留占位符文字
- [ ] 已应用公司品牌色
- [ ] 所有章节标题添加强调条
- [ ] 财务表格格式正确
- [ ] 底部注明数据来源
- [ ] 无文字溢出或截断
- [ ] 投行品质（高盛/摩根士丹利/摩根大通标准）
