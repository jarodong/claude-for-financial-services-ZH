# 材料检查报告格式

## 报告模板

```markdown
# 材料检查报告：[演示名称]

## 摘要
- 总问题数：X
- 严重：X（数字不匹配、事实错误）
- 重要：X（叙事与数据一致性、语言）
- 次要：X（格式）

## 严重问题

### 数字一致性
1. **[问题名称]**（第 X、Y 页）
   - 第 X 页：[数值]
   - 第 Y 页：[数值]
   - 处理建议：[建议]

### 数据与叙事一致性
1. **[问题名称]**（第 X、Y 页）
   - 声称："[引用文本]"
   - 数据显示：[矛盾点]
   - 处理建议：[建议]

## 重要问题

### 语言润色
1. **[问题类型]**（第 X 页）
   - 当前："[引用文本]"
   - 建议："[替换文本]"

## 次要问题

### 格式
1. **[问题类型]**（第 X 页）
   - [描述及修正方法]

## 最终检查清单
- [ ] 数字已勾稽
- [ ] 叙事与数据一致
- [ ] 语言符合投行标准
- [ ] 图表已标注来源
- [ ] 格式统一
```

## 问题严重程度分类

**严重**（客户交付前必须修正）：
- 跨页数字不匹配
- 计算错误
- 事实不准确（姓名、头衔、日期）
- 数据与叙事矛盾

**重要**（应修正）：
- 口语化/非正式用语
- 缺乏具体性的模糊声明
- 术语不一致
- 图表缺少来源

**次要**（润色项）：
- 字体/颜色不一致
- 日期格式差异
- 间距/对齐问题
- 孤立文本
