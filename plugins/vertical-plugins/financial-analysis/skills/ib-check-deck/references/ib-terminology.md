# 投行术语参考

## 口语化 → 专业化替换

| 口语化/非正式 | 投行标准用语 |
|-----------------|-------------|
| "增长很多" | "显著增长" 或 "X% 增长" |
| "利润率还不错" | "有吸引力的利润率" 或 "X% 的利润率" |
| "他们买了这家公司" | "该公司已被收购" |
| "大交易" | "变革性交易" |
| "估值便宜" | "有吸引力的估值" 或 "估值折价" |
| "贵" | "溢价估值" |
| "赚更多钱" | "提升盈利能力" 或 "推动利润率扩张" |
| "越做越大" | "追求增长" 或 "扩展业务" |
| "削减成本" | "实施成本优化" 或 "推动运营效率" |
| "很合适" | "战略契合" 或 "引人注目的战略逻辑" |
| "帮忙" | "支持" 或 "促进" |
| "一堆" | "多个" 或 "大量" |
| "差不多" / "某种程度" | [删除或具体量化] |
| "非常" / "很" | [删除或量化] |
| "大量" | "可观的" 或具体量化 |
| "巨大的" | "重大的" 或具体量化 |
| "基本上" | [删除或精确表述] |

## 应避免的语言模式

- **缩写形式**：不要用 → 不能，不会 → 将不会
- **感叹号**：投行材料中通常不宜使用
- **第一人称**："我们认为..." → "管理层认为..." 或使用被动语态
- **无证据的最高级表述**："行业领先" 需要有支撑数据
- **模糊量词**："一些"、"许多"、"若干" → 具体数字

## 推荐措辞模式

**增长叙事**：
- "已证明 X% 收入 CAGR 的增长记录"
- "在 [期间] 内持续的利润率扩张"
- "具备经验证的有机增长能力"

**市场地位**：
- "[具体细分市场] 中排名第 X 的参与者"
- "[具体产品/服务] 的领先提供商"
- "通过 [具体属性] 实现差异化定位"

**战略逻辑**：
- "由...驱动的引人注目的战略契合"
- "通过...实现有吸引力的价值创造机会"
- "来自 [具体来源] 的 $Xm 协同效应潜力"
