# 客户回顾准备

## 工作流程

### 第一步：客户背景

收集或查询：
- **客户姓名**和家庭成员
- **账户类型**：应税账户、个人养老金账户、个人养老金账户（税后缴费型）、企业年金、信托等
- **总管理资产（AUM）**：跨账户汇总
- **投资政策声明（IPS）**：目标配置、风险承受能力、约束条件
- **人生阶段**：积累期、退休前期、退休期、遗产期
- **上次会议日期**和未完成的行动事项

### 第二步：投资组合业绩

对每个账户和家庭汇总：

| 指标 | 季度 | 年初至今 | 1年 | 3年 | 成立以来 |
|--------|-----|-----|--------|--------|----------------|
| 投资组合收益 | | | | | |
| 基准收益 | | | | | |
| Alpha | | | | | |

**业绩归因：**
- 哪些资产类别/持仓驱动了收益？
- 前3大贡献者和前3大拖累者
- 是否有超大的单持仓影响？

### 第三步：配置审查

当前 vs. 目标配置：

| 资产类别 | 目标 | 当前 | 偏离 | 操作 |
|------------|--------|---------|-------|--------|
| 美国大盘 | | | | |
| 美国中小盘 | | | | |
| 国际发达市场 | | | | |
| 新兴市场 | | | | |
| 固定收益 | | | | |
| 另类投资 | | | | |
| 现金 | | | | |

标记任何超过IPS再平衡阈值（通常3-5%）的偏离。

### 第四步：讨论要点

生成会议议程：

1. **市场概览**（2-3分钟）：简要宏观背景和展望
2. **投资组合业绩**（5分钟）：表现如何？为什么？
3. **配置审查**（5分钟）：是否需要再平衡？
4. **规划更新**（5-10分钟）：
   - 生活变化？（工作、健康、家庭、住房、教育）
   - 收入需求变化？
   - 税务情况更新
   - 遗产规划更新
5. **行动事项**（5分钟）：下次会议前我们要做什么？

### 第五步：主动建议

根据回顾情况，建议：
- 再平衡交易（如偏离超过阈值）
- 税收亏损收割机会
- 现金部署或提取计划
- 个人养老金账户（税后缴费型）转换机会（如适用）
- 受益人更新或遗产规划需求
- 保险审查（人寿、残疾、长期护理）

### 第六步：输出

- 一页纸客户回顾摘要（Word或PDF）
- 含基准的业绩表格
- 配置饼图（当前 vs. 目标）
- 推荐行动事项
- 会议议程

## 重要说明

- 会议前了解客户——回顾上次会议记录
- 以客户关心的内容为主导，而非你想讨论的内容
- 如果业绩不好，直接面对——不要隐藏或粉饰
- 始终以明确的行动事项和带日期的下一步结尾
- 记录会议纪要和IPS的任何变更
- 合规：确保所有材料符合公司政策和监管要求
