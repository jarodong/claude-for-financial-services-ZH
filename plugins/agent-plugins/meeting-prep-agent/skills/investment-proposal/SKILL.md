# 投资建议书

## 工作流程

### 第一步：潜在客户背景

收集：
- **潜在客户姓名**和家庭情况
- **当前状况**：有现有顾问？自主投资？什么促使了这次会面？
- **资产**：预估管理资产规模、账户类型、当前持仓（如有分享）
- **目标**：退休、财富保值、增长、收入、教育、遗产
- **风险承受能力**：保守、稳健、积极（或问卷评分）
- **约束条件**：ESG偏好、集中持仓、流动性需求
- **费用敏感度**：目前支付多少？
- **竞争**：还在考虑哪些机构？

### 第二步：建议书结构

**一、关于我们公司**（1页）
- 公司概况、历史、管理资产规模
- 投资理念（通俗易懂的语言）
- 团队简介（与该客户相关的）
- 客户服务模式（多久见一次面、联系谁）

**二、理解您的需求**（1页）
- 重述他们的目标和担忧——表明您倾听了
- 需求分析中识别的关键规划考虑
- 对他们而言成功的定义

**三、建议投资策略**（2-3页）
- 推荐的资产配置及理由
- 配置如何匹配他们的目标和风险承受能力
- 投资工具（ETF、共同基金、个股、另类投资）
- 税务感知策略（资产配置位置、税收亏损收割）

建议配置：

| 资产类别 | 配置比例 | 工具 | 理由 |
|------------|-----------|---------|-----------|
| | | | |

**四、预期结果**（1-2页）
- 预计增长情景（保守、稳健、乐观）
- 蒙特卡洛达成目标概率
- 收入预测（如退休或收入导向）
- 风险指标（最大回撤、波动率）
- 与当前组合对比（如已知）

**五、费用结构**（1页）
- 顾问费表（阶梯式如适用）
- 底层基金费用
- 总综合成本预估
- 费用与行业平均水平的对比
- 价值主张——费用对应获得的服务

**六、开始合作**（1页）
- 开户流程
- 资产转移时间表
- 过渡计划（如从其他顾问转入）
- 前90天——预期内容
- 所需文件和下一步

### 第三步：定制化

- 根据潜在客户调整语气（企业高管 vs. 小企业主 vs. 退休人士）
- 如有集中持仓，直接回应
- 如与智能投顾比较，强调规划和关系价值
- 如对价格敏感，以总价值和结果为主导，而非仅仅费用

### 第四步：输出

- 带公司品牌的PowerPoint演示文稿（12-15页）
- PDF留底版本
- 一页纸摘要用于后续邮件

## 重要说明

- 建议书应感觉个性化，而非模板化——引用他们的具体情况
- 不要过度销售业绩——设定现实预期并强调流程
- 始终包含免责声明（预测为假设性、过往业绩等）
- 过渡计划很重要——客户担心更换顾问的干扰
- 48小时内跟进，附建议书和明确的下一步
- 向潜在客户展示前必须经合规审查
