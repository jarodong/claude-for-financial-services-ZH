# 客户报告

## 工作流程

### 第一步：报告参数

- **客户姓名**和家庭
- **报告期**：季度、年初至今、年度、自定义区间
- **账户**：全部账户或特定账户
- **基准**：沪深300指数、60/40组合、与IPS匹配的自定义基准
- **公司品牌**：Logo、颜色、免责声明

### 第五步：市场评论

根据客户的金融素养水平定制简要市场概述：
- 本季度市场发生了什么（2-3句话）
- 对投资组合的影响
- 展望和配置逻辑（2-3句话）
- 对零售客户不使用术语；对成熟投资者可以更专业

### 第六步：交易活动摘要

- 期间执行的交易
- 资金存入和提取
- 收到的股息和利息
- 收取的费用
- 再平衡活动

### 第七步：规划备注

- 财务目标进展（退休、教育等）
- 计划变更或建议
- 即将到来的行动事项
- 下次回顾日期

### 第八步：输出

- 带公司品牌的PDF报告（8-12页）
- Word文档便于定制
- Excel数据附件（可选）

**报告结构：**
1. 封面（客户姓名、期间、公司Logo）
2. 执行摘要（1页）
3. 业绩摘要（1-2页）
4. 资产配置概览及图表（1页）
5. 持仓明细（1-2页）
6. 市场评论（1页）
7. 交易活动摘要（1页）
8. 规划备注（1页）
9. 披露和免责声明（1页）

## 重要说明

- 业绩必须按扣除费用后计算，除非客户/合规要求按毛收益计算
- 始终包含适当的免责声明和披露（过往业绩、风险因素）
- 报告应在客户间保持一致——使用标准模板
- 根据客户需求调整详细程度——有些客户想要每只持仓，有些想要一页摘要
- 基准选择很重要——使用IPS中的基准，而非看起来最好的那个
- 新模板首次分发前需经合规审批
