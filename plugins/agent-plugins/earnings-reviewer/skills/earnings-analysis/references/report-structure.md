# 报告结构与模板

本文档提供盈利更新DOCX报告的完整逐页模板和格式要求。

## 完整报告结构

**报告结构：**

## 第1页：盈利摘要

**顶部区域——页眉：**
```
[公司名称] ([股票代码])
[季度] [年份] EARNINGS UPDATE

[当前日期]

评级：[维持/上调/下调] [评级]
股价（截至[date]）：XX.XX元
目标价：[旧 → 新（如有变动），或维持 XXX元]
```

**顶部区域——快速摘要框：**
```
EARNINGS SUMMARY
─────────────────────────────────────────────────
Q[X] [年份] 业绩：[超预期 / 符合预期 / 低于预期]

                报告值      预测值     差异
营收            X,XXX元      X,XXX元     +XXX元 (+X%)
每股收益（调整后）X.XX元       X.XX元      +X.XX元 (+X%)

关键要点：
■ [要点1 - 一句话]
■ [要点2 - 一句话]
■ [要点3 - 一句话]
```

**主内容——投资影响（3-4个要点）：**

使用 ■ 字符，配合**加粗标题**和段落长度的说明：

```
■ **业绩超预期，得益于[分部/地区/产品]的强劲表现，保持积极势头**

Q[X]营收X.XB元超出我们X.XB元的预测X%，超出一致预期X%，
主要由[具体驱动因素]推动。[分部]营收同比增长X%（我们预测为X%），
[分部]增长X%（预测为X%）。管理层强调[具体产品/举措]是关键增长
驱动力，并对前景保持信心。超预期表现验证了[论点要点]，强化了
我们的积极观点。

■ **利润率同比扩张XX个基点，尽管面临[逆风]，展现经营杠杆**

[详细利润率分析段落...]

■ **指引上调/维持/下调——暗示[解读]**

[详细指引分析段落...]

■ **维持[评级]，目标价[上调/不变]至XXX元**

[投资结论段落...]
```

**底部区域——更新预测表格：**

```

                     FY2024E（旧）  FY2024E（新）  变动   FY2025E（新）
营收（百万元）           XX,XXX         XX,XXX         +X%    XX,XXX
营收增长率（%）       X.X%           X.X%           +XXbps X.X%

EBITDA（百万元）         X,XXX          X,XXX          +X%    X,XXX
EBITDA利润率（%）     XX.X%          XX.X%          +XXbps XX.X%
每股收益（调整后）(元)  X.XX           X.XX           +X%    X.XX
市盈率（x）           XX.Xx          XX.Xx          -X%    XX.Xx

注："E"=预测值。旧预测来自[此前报告日期]。
```

- 同比和环比趋势
- 与指引比较（如有）

## 格式要求

### 1. 第1页要求
- 清晰评级（维持跑赢大市、上调至买入等）
- 突出展示更新目标价
- 摘要表格含新旧预测
- 3-4个使用■字符的段落级要点

### 2. 所有表格要求
- 底部有来源行
- 清晰列标题
- 标题行有底色

### 3. 所有图表要求
- 上方有"Figure X - [标题]"说明
- 下方有"Source: [来源]"行
- 专业样式

### 4. 年份标注
- A表示实际值（Q3'24A）
- E表示预测值（Q4'24E）

### 5. 写作风格
- 数字引领（"营收增长15%至12亿元"而非"强劲的营收增长"）
- 使用"vs."而非"versus"
- 直接简洁
- 聚焦新信息

### 6. 超链接要求 ⭐⭐⭐
- 所有URL在Word中必须为可点击超链接
- 蓝色、带下划线文本，Ctrl+Click可打开
- 显示文本有意义（非原始URL）
- 每个来源引用在适用处应附可点击链接
- 无纯文本URL——始终格式化为超链接
