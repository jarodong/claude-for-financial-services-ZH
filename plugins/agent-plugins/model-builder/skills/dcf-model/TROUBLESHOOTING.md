# DCF 模型故障排除指南

**何时阅读此文件：** 当 recalc.py 显示错误，或估值结果看起来不合理，或情景选择器工作异常时。

## 模型返回错误值

### #REF! 错误
- 通常由插入标题后公式引用了错误行导致
- 解决方案：使用正确的行引用重新构建，或从头开始按布局规划操作
- 预防措施：在写公式之前定义所有行位置

### #DIV/0! 错误
- 除以零或空单元格
- 解决方案：添加 IF 语句处理零值：`=IF([Divisor]=0,0,[Numerator]/[Divisor])`

### #VALUE! 错误
- 计算中的数据类型错误（文本而非数字）
- 解决方案：验证所有输入格式为数字

## 估值看起来不合理

### 隐含价格过高
- 检查终值是否超过 EV 的 80%
- 验证终值增长率 < WACC
- 审查增长假设是否现实
- 考虑利润率假设是否过于乐观

### 隐含价格过低
- 验证净债务与净现金是否正确
- 检查 WACC 是否过高
- 审查预测是否过于保守
- 考虑终值增长率是否过低

## 情景选择器不工作

### 切换情景时汇总列未更新
- 验证情景选择器单元格包含 1、2 或 3
- 检查 INDEX/OFFSET 公式引用了正确的行范围和选择器单元格
- 确保对选择器使用了绝对引用（$B$6）
- 通过手动更改选择器单元格并验证预测值是否更新来进行测试
