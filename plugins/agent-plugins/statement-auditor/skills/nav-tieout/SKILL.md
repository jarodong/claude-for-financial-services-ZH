# NAV 核对

给定已生成的 LP 对账单和当期 NAV 包（通过 nav MCP），独立重新计算 LP 资本账户并逐行比较。

> **已生成的对账单是被测试对象。** NAV 包为数据源。

## 重新计算 LP 资本账户

```
期初资本（上期末对账单余额）
  + 缴入（本期已缴资本催缴款）
  − 分配（现金 + 实物）
  + 已分配净收益 /（亏损）
      = LP 占比 ×（已实现 + 未实现损益 − 管理费 − 基金费用）
  − 业绩报酬分配（如本期已确定）
期末资本
```

从 NAV 包中提取各输入项：LP 承诺占比、基金层面损益组成部分、费用及开支合计、瀑布分配结果。

## 逐行比较

对账单中的每一行，与重新计算值比较。容差：`0.01`。对每处不一致，注明由哪个输入项导致（如"已分配损益存在差异——对账单使用 12.40% 持股比例，NAV 包在第一季度转让后显示 12.38%"）。

## 附加检查

- 本对账单期末资本 = 下期对账单初稿期初资本（如有）。
- 所有 LP 期末资本合计 = 基金 NAV（在四舍五入范围内）。
- 承诺额、未缴额、可回拨额与承诺登记簿一致。

## 输出

逐行通过/未通过结果、重新计算值与对账单对照值，以及问题标记清单。不得修改对账单——发布方在审核后根据标记进行处理。
