# 金融套件翻译专业名词表

> 翻译时参照此表。术语处理分三档：固定保留英文缩写/英文术语 + 中文释义、首次出现中文 + 英文括注、直接中文化。

---

## 使用规则

1. 固定保留英文缩写/英文术语 + 中文释义：适用于 DCF、LBO、Comps、Precedents、WACC、IRR、NAV、GL、KYC、AML、TLH 等中文读者常见但仍需解释的专业缩写或投行业务术语。首次出现和表格入口统一写作“英文（中文释义）”，如 DCF（折现现金流估值法）。
2. 首次出现中文 + 英文括注，后续用中文：适用于中文金融语境已有稳定表达、但需要保留英文对照的术语，如投决会备忘录（Investment Committee Memo）、管理层演示（Management Presentation）、数据室（Data Room）。
3. 直接中文化：适用于利润表、资产负债表、现金流量表、营业收入、净利润等已高度中文化的财务报表和会计科目。
4. 技术产品名保留英文名，首次出现加中文释义：Agent（智能体）、Skill（技能）、Plugin（插件）、MCP（模型上下文协议）、Claude Code 等原则上保留英文名，首次出现时加中文解释。
